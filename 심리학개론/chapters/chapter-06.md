# Chapter 06: 학습과 기억

---

## 핵심 한 줄

> "학습 = 경험으로 인한 행동 변화. 고전적(파블로프) vs 조작적(스키너) 조건형성 구분!"

---

## 1. 학습의 정의

```
학습 = 경험에 의한 비교적 영속적인 행동 변화

학습 아닌 것:
├── 성숙 (유전적 변화)
├── 피로 (일시적)
└── 약물/질병 (생리적 변화)
```

---

## 2. 고전적 조건형성 (파블로프) ★★★

### 실험: 개의 침 분비
```
실험 전:
음식(US) → 침 분비(UR)
종소리 → 반응 없음

조건형성:
종소리 + 음식 → 침 분비 (반복)

조건형성 후:
종소리(CS) → 침 분비(CR)

용어:
US = 무조건 자극 (음식)
UR = 무조건 반응 (침)
CS = 조건 자극 (종소리)
CR = 조건 반응 (종소리에 침)
```

### 원리
```
1. 획득: CS-US 연합 학습
2. 소거: CS만 주면 CR 사라짐
3. 자발적 회복: 소거 후 CR 다시 나타남
4. 자극 일반화: 비슷한 자극에도 반응
5. 자극 변별: 차이 구분
```

### 예시
```
일상 예:
├── 레몬만 봐도 침 (시각→침)
├── 병원 냄새 → 불안 (냄새→불안)
└── 특정 노래 → 추억 (소리→정서)

공포증 형성:
개에게 물림(US) → 공포(UR)
개(CS) → 공포(CR)
```

---

## 3. 조작적 조건형성 (스키너) ★★★

### 핵심 원리
```
행동의 결과가 미래 행동을 결정

행동 → 좋은 결과 → 행동 증가
행동 → 나쁜 결과 → 행동 감소
```

### 강화와 처벌
```
┌─────────────────────────────────────────────────┐
│            │ 제시 (+)      │ 제거 (-)        │
├─────────────────────────────────────────────────┤
│ 행동 증가  │ 정적 강화     │ 부적 강화       │
│ (강화)     │ 칭찬, 보상    │ 잔소리 멈춤     │
├─────────────────────────────────────────────────┤
│ 행동 감소  │ 정적 처벌     │ 부적 처벌       │
│ (처벌)     │ 때림, 벌금   │ 용돈 삭감       │
└─────────────────────────────────────────────────┘

암기 팁:
정적 = 뭔가 줌
부적 = 뭔가 뺌
강화 = 행동↑
처벌 = 행동↓
```

### 강화 스케줄
```
연속 강화: 매번 강화 (빠른 학습, 빠른 소거)

간헐 강화: 가끔 강화 (느린 학습, 느린 소거)
├── 고정비율: N회마다 (공장 성과급)
├── 변동비율: 평균 N회 (도박, 슬롯머신) ★가장 강력
├── 고정간격: N분마다 (월급)
└── 변동간격: 평균 N분 (낚시)
```

## 4. 고전적 vs 조작적 비교

```
┌────────────────────────────────────────────────┐
│          │ 고전적 조건형성  │ 조작적 조건형성 │
├────────────────────────────────────────────────┤
│ 학자     │ 파블로프         │ 스키너          │
├────────────────────────────────────────────────┤
│ 반응     │ 자동적 (반사)    │ 자발적 (행동)   │
├────────────────────────────────────────────────┤
│ 자극순서 │ 자극→반응        │ 행동→결과       │
├────────────────────────────────────────────────┤
│ 핵심     │ 자극의 연합      │ 행동의 결과     │
├────────────────────────────────────────────────┤
│ 예시     │ 종→침            │ 공부→칭찬       │
└────────────────────────────────────────────────┘
```

## 5. 관찰학습 (반두라)

```
관찰학습 = 타인을 보고 배움 (모방학습, 대리학습)

보보인형 실험:
├── 공격적 모델 본 아이 → 공격적 행동
└── 비공격적 모델 → 덜 공격적

4단계:
1. 주의: 모델에 주목
2. 파지: 기억
3. 재생: 행동으로 옮김
4. 동기화: 강화 기대

자기효능감:
└── "나도 할 수 있다"는 믿음
└── 반두라의 중요 개념
```

## 6. 기억

### 기억의 3단계 모형
```
┌─────────┐    ┌─────────┐    ┌─────────┐
│ 감각기억 │ → │ 단기기억 │ → │ 장기기억 │
└─────────┘    └─────────┘    └─────────┘
   ↓             ↓             ↓
 ~1초         15-30초         무한

감각기억:
├── 아이콘 기억 (시각)
└── 에코 기억 (청각)

단기기억:
├── 용량: 7±2개 (마법의 숫자)
├── 청킹으로 확장 가능
└── 시연하면 장기기억으로

장기기억:
├── 무한 용량
├── 영구 저장
└── 인출 문제
```

### 장기기억의 종류
```
장기기억
├── 외현기억 (선언적, 의식적)
│   ├── 의미기억: 지식, 사실
│   └── 일화기억: 개인 경험
│
└── 암묵기억 (비선언적, 무의식적)
    ├── 절차기억: 기술 (자전거)
    ├── 조건화
    └── 점화
```

### 망각 이론
```
1. 쇠퇴이론: 시간 지나면 흐려짐
2. 간섭이론:
   ├── 순행간섭: 이전 학습이 새 학습 방해
   └── 역행간섭: 새 학습이 이전 기억 방해
3. 인출실패: 저장됐지만 못 꺼냄

에빙하우스 망각곡선:
└── 처음에 급격히 망각, 이후 완만
```

## 핵심 암기 포인트

### 1. 고전적 조건형성
```
파블로프, 자동반응, 자극연합
US→UR, CS→CR
```

### 2. 조작적 조건형성
```
스키너, 자발행동, 결과가 행동 결정
정적 강화(+주기), 부적 강화(-제거), 처벌(감소)
```

### 3. 관찰학습
```
반두라, 모방, 자기효능감
```

### 4. 기억 3단계
```
감각(1초) → 단기(30초, 7±2) → 장기(무한)
```

### 5. 망각
```
쇠퇴, 간섭(순행/역행), 인출실패
```

## 예상 문제

### Q1. 고전적 조건형성에서 CS는?
→ **조건 자극** (예: 종소리)

### Q2. 정적 강화와 부적 강화의 공통점은?
→ 둘 다 **행동을 증가**시킴

### Q3. 도박중독과 관련된 강화 스케줄은?
→ **변동비율 강화** (언제 당첨될지 모름)

### Q4. 단기기억의 용량은?
→ **7±2개** (마법의 숫자 7)

### Q5. "나도 할 수 있다"는 믿음을 뭐라 하나?
→ **자기효능감** (반두라)
