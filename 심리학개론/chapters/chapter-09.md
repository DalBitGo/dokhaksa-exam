## 1. 정신분석 이론 (프로이트) ★★★

### 의식의 구조
```
       ┌─────────────────────┐
       │      의식           │ ← 알고 있는 것
       ├─────────────────────┤
       │     전의식          │ ← 노력하면 떠올림
       ├─────────────────────┤
       │                     │
       │     무의식          │ ← 접근 불가, 본능
       │                     │    억압된 욕구
       └─────────────────────┘

빙산 비유:
├── 의식: 수면 위 (작음)
└── 무의식: 수면 아래 (큼)
```

### 성격 구조 (이드, 자아, 초자아)
```
┌─────────────────────────────────────────┐
│ 구조    │ 원리      │ 기능              │
├─────────────────────────────────────────┤
│ 이드    │ 쾌락원리  │ 본능, 욕구 충족   │
│ (Id)    │           │ "지금 당장!"      │
├─────────────────────────────────────────┤
│ 자아    │ 현실원리  │ 현실적 충족       │
│ (Ego)   │           │ "어떻게 하면?"    │
├─────────────────────────────────────────┤
│ 초자아  │ 도덕원리  │ 양심, 이상        │
│(Superego)│          │ "이래야 한다"     │
└─────────────────────────────────────────┘

갈등:
이드(하고싶어) ↔ 초자아(하면 안돼)
         ↓
   자아가 중재
```

### 방어기제 ★★
```
방어기제 = 불안 감소를 위한 무의식적 전략

┌──────────────────────────────────────────────────┐
│ 기제      │ 설명                  │ 예시        │
├──────────────────────────────────────────────────┤
│ 억압      │ 불쾌한 것 의식서 밀어냄│ 트라우마 망각│
│ 부정      │ 현실 부인              │ "그럴리없어"│
│ 투사      │ 내 감정을 남탓         │ 내가 미워→걔가│
│ 합리화    │ 그럴듯한 핑계          │ 여우와 포도 │
│ 반동형성  │ 반대로 행동            │ 좋아하는데 무시│
│ 퇴행      │ 이전 단계로 돌아감     │ 아기처럼 행동│
│ 승화      │ 욕구를 사회적으로 표현 │ 공격성→스포츠│
│ 전치      │ 대상 바꿈              │ 상사한테 혼남→│
│           │                       │ 부하한테 화냄│
└──────────────────────────────────────────────────┘
```

### 심리성적 발달단계
```
구강기 (0~1세): 입, 빨기
항문기 (1~3세): 배변훈련
남근기 (3~6세): 성기 관심, 오이디푸스
잠복기 (6~12세): 성적 욕구 잠복
생식기 (12세~): 성숙한 성적 관심
```

## 2. 분석심리학 (융)

### 융의 개념
```
개인무의식: 개인의 억압된 것
집단무의식: 인류 공통의 무의식 ← 융의 독자적 개념

원형 (Archetype):
├── 페르소나: 사회적 가면
├── 그림자: 어두운 면
├── 아니마/아니무스: 내면의 이성
└── 자기: 전체 통합
```

---

## 3. 인본주의 이론

### 매슬로우 욕구위계론
```
        /\
       /  \  자아실현
      /────\
     /  자존  \
    /──────────\
   /    소속     \
  /────────────────\
 /      안전         \
/────────────────────────\
       생리적 욕구

아래→위로 충족
결핍욕구 (1-4단계) vs 성장욕구 (5단계)
```

### 로저스의 인간중심이론
```
핵심 개념:
├── 자아개념: 자신에 대한 지각
├── 이상적 자아: 되고 싶은 모습
├── 현실 자아: 실제 모습
└── 일치 ↔ 불일치

조건 없는 긍정적 존중:
└── 무조건적 수용
└── 자아실현의 조건

충분히 기능하는 사람:
├── 경험에 개방
├── 현재를 삶
└── 자신을 신뢰
```

## 4. 특성 이론

### 올포트
```
추기경 특성: 지배적, 삶 전체 영향 (소수)
중심 특성: 일반적 특성 (5-10개)
이차 특성: 상황적 특성
```

### Big 5 (5요인 모델) ★
```
OCEAN:
O - Openness (개방성): 새로운 경험
C - Conscientiousness (성실성): 자기조절
E - Extraversion (외향성): 사교성
A - Agreeableness (우호성): 협조
N - Neuroticism (신경증): 불안, 정서불안정

암기: "오션에서 수영하는 성격"
```


## 예상 문제

### Q1. 프로이트의 성격구조 중 현실원리를 따르는 것은?
→ **자아 (Ego)**

### Q2. "좋아하면서 무관심한 척"하는 방어기제는?
→ **반동형성**

### Q3. 융의 "집단무의식"이란?
→ **인류 공통의 무의식**, 원형 포함

### Q4. 매슬로우 욕구위계의 최상위는?
→ **자아실현**

### Q5. Big 5의 "N"은?
→ **신경증 (Neuroticism)** - 정서적 불안정
