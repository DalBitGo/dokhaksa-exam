# Chapter 02: 행동의 생리적 기초

---

## 핵심 한 줄

> "뇌 = 행동의 기관. 뉴런이 전기신호로 소통하고, 대뇌 각 부분이 다른 기능 담당"

---

## 1. 뉴런 (신경세포)

### 뉴런의 구조
```
       수상돌기 (Dendrite)
         ┃┃┃
         ▼▼▼
    ┌──────────┐
    │  세포체   │ ← 신호 수신, 처리
    │  (Soma)  │
    └────┬─────┘
         │
    ┌────┴─────┐
    │  축삭    │ ← 신호 전달 (긴 섬유)
    │ (Axon)   │
    └────┬─────┘
         │
    ▼▼▼▼▼▼▼▼
    축삭종말
   (시냅스로 연결)

수상돌기: 신호 받음
세포체: 신호 통합
축삭: 신호 전달
시냅스: 다음 뉴런으로 전달
```

### 신경 전달 과정
```
1. 수상돌기가 신호 수신
2. 세포체에서 신호 통합
3. 역치 넘으면 활동전위 발생
4. 축삭 따라 전기신호 이동
5. 시냅스에서 신경전달물질 분비
6. 다음 뉴런의 수상돌기가 수신

┌─────┐        ┌─────┐
│뉴런A│──시냅스──│뉴런B│
└─────┘   ↑    └─────┘
     신경전달물질
```

### 신경전달물질
```
┌───────────────────────────────────────────┐
│ 물질          │ 기능                       │
├───────────────────────────────────────────┤
│ 아세틸콜린    │ 근육 수축, 기억            │
│ 도파민        │ 쾌락, 보상, 운동           │
│ 세로토닌      │ 기분, 수면, 식욕           │
│ 노르에피네프린│ 각성, 주의                 │
│ GABA         │ 억제 (불안 감소)            │
│ 글루타메이트  │ 흥분, 학습                 │
│ 엔도르핀      │ 통증 억제, 쾌감            │
└───────────────────────────────────────────┘

암기 팁:
도파민 ↓ → 파킨슨병
도파민 ↑ → 조현병
세로토닌 ↓ → 우울증
```

## 2. 신경계의 구조

### 신경계 체계
```
신경계
├── 중추신경계 (CNS)
│   ├── 뇌 (Brain)
│   └── 척수 (Spinal Cord)
│
└── 말초신경계 (PNS)
    ├── 체성신경계 (수의적)
    │   └── 골격근 제어
    │
    └── 자율신경계 (불수의적)
        ├── 교감신경계: 흥분 (싸움 or 도주)
        └── 부교감신경계: 이완 (휴식 & 소화)
```

### 교감 vs 부교감
```
교감신경 (Emergency)    부교감신경 (Rest)
─────────────────────────────────────
동공 확대               동공 축소
심박수 증가             심박수 감소
소화 억제               소화 촉진
에너지 방출             에너지 저장

예:
호랑이 만남 → 교감신경 활성화
밥 먹고 쉼 → 부교감신경 활성화
```

## 3. 대뇌의 구조

### 대뇌피질 4엽
```
         ┌─────────────────┐
         │    전두엽       │ ← 계획, 판단, 인격
         │  (Frontal)     │
         ├─────────────────┤
         │    두정엽       │ ← 촉각, 공간감각
         │  (Parietal)    │
    ┌────┴─────────────────┴────┐
    │        측두엽              │ ← 청각, 언어이해
    │      (Temporal)           │
    └───────────────────────────┘
              후두엽              ← 시각
            (Occipital)

암기: 전-계획 / 두-촉각 / 측-청각 / 후-시각
```

### 대뇌 각 영역
```
전두엽 기능:
├── 계획, 의사결정
├── 운동 제어 (운동피질)
├── 성격, 충동 조절
└── 브로카 영역 (언어 생성)

두정엽 기능:
├── 촉각 처리 (체감각피질)
├── 공간 인식
└── 주의 집중

측두엽 기능:
├── 청각 처리
├── 기억 형성 (해마 연결)
└── 베르니케 영역 (언어 이해)

후두엽 기능:
└── 시각 처리
```

### 언어 영역
```
브로카 영역 (전두엽)
├── 언어 생성, 말하기
└── 손상 → 브로카 실어증 (말 못함, 이해는 OK)

베르니케 영역 (측두엽)
├── 언어 이해
└── 손상 → 베르니케 실어증 (말은 되지만 의미 없음)
```

### 변연계
```
변연계 = 정서와 기억의 중추

구성:
├── 해마 (Hippocampus)
│   └── 기억 형성 (단기→장기)
│
├── 편도체 (Amygdala)
│   └── 정서, 특히 공포
│
├── 시상하부 (Hypothalamus)
│   └── 항상성 (배고픔, 갈증, 체온, 성욕)
│
└── 시상 (Thalamus)
    └── 감각 중계소 (후각 제외)
```

## 4. 내분비계

### 내분비계란?
```
내분비계 = 호르몬을 분비하는 선(gland)들

호르몬 vs 신경전달물질:
├── 호르몬: 혈액으로 이동, 느리지만 지속
└── 신경전달물질: 시냅스에서, 빠르지만 짧음
```

### 주요 내분비선
```
┌───────────────────────────────────────────┐
│ 선           │ 호르몬       │ 기능        │
├───────────────────────────────────────────┤
│ 뇌하수체     │ 성장호르몬   │ 성장        │
│ (마스터선)   │ 각종 조절    │ 다른 선 조절│
├───────────────────────────────────────────┤
│ 갑상선       │ 티록신       │ 대사 조절   │
├───────────────────────────────────────────┤
│ 부신        │ 아드레날린   │ 스트레스    │
│              │ 코르티솔     │             │
├───────────────────────────────────────────┤
│ 췌장        │ 인슐린       │ 혈당 조절   │
├───────────────────────────────────────────┤
│ 성선        │ 에스트로겐   │ 성 발달     │
│ (난소/고환)  │ 테스토스테론 │             │
└───────────────────────────────────────────┘
```

## 핵심 암기 포인트

### 1. 뉴런 구조
```
수상돌기(수신) → 세포체(통합) → 축삭(전달) → 시냅스(분비)
```

### 2. 신경전달물질
```
도파민: 쾌락, 보상 (↓파킨슨, ↑조현병)
세로토닌: 기분 (↓우울증)
```

### 3. 자율신경계
```
교감: 흥분 (Fight or Flight)
부교감: 이완 (Rest & Digest)
```

### 4. 대뇌 4엽
```
전두엽-계획, 두정엽-촉각, 측두엽-청각, 후두엽-시각
```

### 5. 변연계
```
해마-기억, 편도체-공포, 시상하부-항상성
```

## 예상 문제

### Q1. 뉴런에서 신호를 받는 부분은?
→ **수상돌기**

### Q2. 우울증과 관련된 신경전달물질은?
→ **세로토닌** (부족 시 우울)

### Q3. "싸움 또는 도주" 반응을 담당하는 것은?
→ **교감신경계**

### Q4. 언어 생성을 담당하는 영역은?
→ **브로카 영역** (전두엽)

### Q5. 기억 형성에 중요한 변연계 구조는?
→ **해마**
