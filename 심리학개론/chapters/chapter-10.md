# Chapter 10: 이상행동과 정신장애

---

## 핵심 한 줄

> "이상행동 판단기준 4가지 + 불안장애/기분장애/정신분열 핵심 증상 암기!"

---

## 1. 이상행동의 정의 ★

### 판단 기준 4가지
```
1. 통계적 기준: 평균에서 벗어남
   한계: 천재도 비정상?

2. 사회적 기준: 사회규범 위반
   한계: 문화마다 다름

3. 주관적 기준: 본인이 고통
   한계: 자각 못하는 경우

4. 적응적 기준: 일상 기능 손상 ★
   가장 중요!
```

### DSM-5
```
미국정신의학회 진단분류체계
현재 가장 많이 사용
```

---

## 2. 불안장애 ★★

### 범불안장애 (GAD)
```
만성적, 광범위한 불안
특정 대상 없이 걱정
신체 증상: 긴장, 불면
```

### 공황장애
```
갑작스러운 극심한 공포
심장 두근거림, 호흡곤란
광장공포증 동반 가능
```

### 공포증
```
특정 대상/상황에 대한 과도한 두려움

특정 공포증: 동물, 높은 곳 등
사회공포증: 사회적 상황 두려움
광장공포증: 탈출 어려운 장소
```

### 강박장애 (OCD)
```
강박사고: 반복적 침투사고
강박행동: 의례적 행동

예: 손 씻기, 확인하기
```

### PTSD (외상 후 스트레스 장애)
```
외상 사건 후 발생
플래시백, 악몽
회피 행동, 과각성
```

---

## 3. 기분장애 ★★

### 주요우울장애
```
증상:
├── 우울한 기분
├── 흥미/즐거움 상실
├── 식욕/수면 변화
├── 피로감
├── 무가치감, 죄책감
└── 자살 사고

기간: 2주 이상
```

### 양극성 장애 (조울증)
```
조증 삽화 + 우울 삽화

조증:
├── 기분 고양
├── 에너지 증가
├── 수면 욕구 감소
├── 과대망상
└── 충동적 행동
```

---

## 4. 조현병 (정신분열증) ★★

### 양성 증상
```
환각: 없는 것을 지각 (환청 가장 흔함)
망상: 잘못된 믿음
├── 피해망상: 누군가 해치려 함
├── 과대망상: 특별한 능력/지위
└── 관계망상: 모든 것이 자신과 관련

와해된 언어/행동
```

### 음성 증상
```
정서적 둔마: 감정 표현 감소
무의욕증: 의욕 저하
사회적 위축
언어 빈곤
```

---

## 5. 성격장애 ★

### A군 (기이형)
```
편집성: 의심, 불신
분열성: 사회적 무관심
분열형: 기이한 생각/행동
```

### B군 (극적/감정적)
```
반사회성: 타인 권리 무시 ★
경계성: 불안정한 관계/자아
연극성: 과도한 감정/관심 추구
자기애성: 자기 과대평가
```

### C군 (불안/두려움)
```
회피성: 거절 두려움
의존성: 과도한 의존
강박성: 완벽주의, 통제
```

---

## 6. 이상행동의 원인

### 생물학적 관점
```
신경전달물질 불균형
├── 도파민 ↑: 조현병
├── 세로토닌 ↓: 우울증
└── GABA ↓: 불안장애

유전적 요인
뇌 구조 이상
```

### 심리학적 관점
```
정신분석: 무의식적 갈등
행동주의: 학습된 행동
인지: 인지적 왜곡/오류
인본주의: 자아실현 좌절
```

### 통합 모델
```
취약성-스트레스 모델:
소인(취약성) + 스트레스 → 장애 발현
```

---

## 7. 치료

### 생물학적 치료
```
약물치료:
├── 항정신병약 (조현병)
├── 항우울제 (우울증)
├── 항불안제 (불안장애)
└── 기분안정제 (양극성)

전기경련요법 (ECT)
```

### 심리치료
```
정신분석: 무의식 탐색
행동치료: 체계적 둔감화, 토큰 경제
인지치료: 인지적 왜곡 수정
인본주의: 무조건적 긍정적 존중
```

## 핵심 암기 포인트

### 1. 이상행동 4기준
```
통계적, 사회적, 주관적, 적응적
(가장 중요: 적응적 기준)
```

### 2. 불안장애 종류
```
범불안, 공황, 공포증, 강박, PTSD
```

### 3. 조현병 증상
```
양성: 환각, 망상 (있는 것)
음성: 둔마, 무의욕 (없어진 것)
```

### 4. 신경전달물질
```
도파민↑: 조현병
세로토닌↓: 우울증
```

## 예상 문제

### Q1. 이상행동 판단기준 중 가장 중요한 것은?
→ **적응적 기준** (일상 기능 손상)

### Q2. 조현병의 양성 증상이 아닌 것은?
→ **무의욕증** (음성 증상)

### Q3. 도파민 과잉과 관련된 장애는?
→ **조현병**

### Q4. 조증과 우울이 교대로 나타나는 장애는?
→ **양극성 장애** (조울증)
