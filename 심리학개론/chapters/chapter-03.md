## 1. 발달의 개념

### 발달이란?
```
발달 = 수정에서 죽음까지 일어나는 변화

성숙: 유전적 프로그램에 의한 변화
학습: 경험에 의한 변화
발달: 성숙 + 학습
```

### 발달의 원리
```
1. 연속성: 점진적으로 일어남
2. 결정적 시기: 특정 시기에 특정 발달
3. 개인차: 사람마다 속도 다름
4. 분화와 통합: 단순→복잡
```

### 유전 vs 환경
```
Nature (유전) vs Nurture (환경)

현대 관점:
├── 둘 다 중요
├── 상호작용
└── 유전이 잠재력, 환경이 실현
```

## 2. 피아제의 인지발달 (가장 중요!)

### 핵심 개념
```
도식 (Schema): 세상을 이해하는 틀

동화 (Assimilation): 기존 도식에 맞춤
조절 (Accommodation): 도식을 수정

예: 네발 달린 것 = 강아지 (도식)
├── 고양이 봄 → "강아지!" (동화)
└── 아니라고 알려줌 → 새 도식 형성 (조절)
```

### 인지발달 4단계 ★★★
```
┌─────────────────────────────────────────────────┐
│ 단계        │ 나이    │ 특징                    │
├─────────────────────────────────────────────────┤
│ 감각운동기  │ 0~2세   │ 감각과 운동으로 탐색    │
│             │         │ 대상영속성 발달         │
├─────────────────────────────────────────────────┤
│ 전조작기    │ 2~7세   │ 언어 발달, 상징 사용    │
│             │         │ 자기중심성, 물활론      │
│             │         │ 보존개념 미발달         │
├─────────────────────────────────────────────────┤
│ 구체적 조작기│ 7~11세  │ 논리적 사고 가능        │
│             │         │ 보존개념 획득           │
│             │         │ 구체적 상황에서만       │
├─────────────────────────────────────────────────┤
│ 형식적 조작기│ 11세~   │ 추상적 사고 가능        │
│             │         │ 가설 설정, 연역적 사고  │
└─────────────────────────────────────────────────┘
```

### 주요 개념 상세
```
대상영속성 (감각운동기):
└── 눈앞에서 사라져도 존재한다는 것 앎
└── 까꿍 놀이가 재밌는 이유

자기중심성 (전조작기):
└── 자기 관점에서만 봄
└── "내가 보면 다른 사람도 보인다"

물활론 (전조작기):
└── 무생물도 살아있다고 생각
└── "인형이 아파해"

보존개념 (구체적 조작기):
└── 모양 바뀌어도 양은 같다
└── 물컵 실험 (좁은 컵→넓은 컵)
```

## 3. 에릭슨의 심리사회적 발달 8단계 ★★

```
┌─────────────────────────────────────────────────────┐
│ 단계 │ 시기        │ 과제          │ 주요 관계     │
├─────────────────────────────────────────────────────┤
│ 1   │ 영아기      │ 신뢰 vs 불신  │ 양육자       │
│ 2   │ 초기 아동기 │ 자율 vs 수치  │ 부모         │
│ 3   │ 유희기      │ 주도 vs 죄책  │ 가족         │
│ 4   │ 학령기      │ 근면 vs 열등  │ 학교, 이웃   │
│ 5   │ 청소년기    │ 정체감 vs 혼란│ 또래         │
│ 6   │ 성인 초기   │ 친밀 vs 고립  │ 친구, 연인   │
│ 7   │ 성인 중기   │ 생산 vs 침체  │ 가정, 직장   │
│ 8   │ 노년기      │ 통합 vs 절망  │ 인류         │
└─────────────────────────────────────────────────────┘
```

### 핵심 과제
```
청소년기 (정체감 vs 역할 혼란):
├── "나는 누구인가?"
├── 정체감 형성 실패 → 역할 혼란
└── 심리사회적 유예 기간

성인 초기 (친밀감 vs 고립):
├── 친밀한 관계 형성
└── 실패 → 고립감

성인 중기 (생산성 vs 침체):
├── 다음 세대 양육, 사회 기여
└── 실패 → 침체감
```

## 4. 콜버그의 도덕성 발달 ★

### 3수준 6단계
```
┌──────────────────────────────────────────────────┐
│ 수준          │ 단계              │ 특징        │
├──────────────────────────────────────────────────┤
│ 전인습 수준   │ 1. 처벌 회피      │ 벌 피하려고 │
│ (아동)        │ 2. 상호 이익      │ 이익 교환   │
├──────────────────────────────────────────────────┤
│ 인습 수준     │ 3. 좋은 아이      │ 착한 사람   │
│ (청소년~성인) │ 4. 사회 질서      │ 법과 질서   │
├──────────────────────────────────────────────────┤
│ 후인습 수준   │ 5. 사회계약       │ 합의된 규칙 │
│ (소수 성인)   │ 6. 보편적 원리    │ 양심, 정의  │
└──────────────────────────────────────────────────┘
```

### 하인츠 딜레마
```
상황: 아내가 아픈데 약이 너무 비쌈. 훔칠까?

1단계: 훔치면 안 됨 (감옥 가니까)
2단계: 훔쳐도 됨 (아내가 나중에 보답)
3단계: 훔쳐도 됨 (좋은 남편이니까)
4단계: 훔치면 안 됨 (법을 어기니까)
5단계: 상황에 따라 (법보다 생명)
6단계: 훔쳐도 됨 (생명권이 재산권보다 우선)
```

## 5. 애착 (보울비, 에인스워스)

### 애착 유형
```
낯선 상황 실험 (에인스워스):

안정 애착 (60%):
├── 엄마 있으면 탐색
├── 분리 시 불안
└── 재결합 시 안정

불안정 애착:
├── 회피형: 무관심
├── 저항형: 분리 후에도 화남
└── 혼란형: 일관성 없음
```
