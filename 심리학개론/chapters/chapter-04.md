# Chapter 04: 동기와 정서

---

## 핵심 한 줄

> "동기 = 행동의 원인, 정서 = 느낌의 반응. 매슬로 욕구위계, 정서이론 3개 암기!"

---

## 1. 동기의 개념

### 동기란?
```
행동을 시작하고, 방향 짓고, 유지하게 하는 내적 상태
= 왜 행동하는가?
```

### 동기의 구성요소
```
활성화: 행동 시작
방향: 목표 지향
강도: 노력의 정도
지속: 행동 유지
```

---

## 2. 동기이론 ★★

### 본능이론
```
맥두걸: 본능이 행동 유발
한계: 설명적, 예측 불가
```

### 추동이론 (Hull)
```
추동 = 생리적 욕구에서 발생
항상성: 균형 유지하려는 경향
한계: 호기심, 탐색 행동 설명 불가
```

### 유인이론
```
외적 보상이 행동 유발
유인가 (incentive value)
```

### 각성이론
```
최적 각성 수준 유지 추구
너무 높거나 낮으면 불쾌
```

---

## 3. 매슬로의 욕구위계 ★★★

### 5단계 피라미드
```
         /\
        /  \
       / 자아 \       5. 자아실현
      / 실현  \
     /────────\
    /   존경    \     4. 존경 욕구
   /────────────\
  /     소속      \    3. 소속/애정 욕구
 /────────────────\
/     안전          \   2. 안전 욕구
──────────────────────
      생리             1. 생리적 욕구

결핍욕구 (1-4): 충족되지 않으면 불안
성장욕구 (5): 자아실현
```

### 각 단계 설명
```
1. 생리적 욕구: 음식, 물, 수면, 성
2. 안전 욕구: 위험 회피, 안정
3. 소속/애정: 사랑, 소속감
4. 존경 욕구: 자존감, 인정
5. 자아실현: 잠재력 발휘
```

---

## 4. 내재적/외재적 동기 ★

### 구분
```
내재적 동기: 활동 자체가 보상
├── 흥미, 호기심, 도전
└── 예: 재미로 책 읽기

외재적 동기: 외적 보상/처벌
├── 돈, 칭찬, 성적
└── 예: 상금 때문에 대회 참가
```

### 과잉정당화 효과 ★
```
내재적으로 즐기던 활동에
외적 보상 → 내재적 동기 ↓

예: 그림 그리기 좋아하던 아이
   → 상금 받음 → 상금 없으면 안 그림
```

## 5. 정서의 개념

### 정서란?
```
특정 자극에 대한 복합적 반응
= 주관적 느낌 + 생리적 변화 + 행동
```

### 정서의 구성요소
```
주관적 경험: 느낌
생리적 반응: 심박, 발한
행동적 표현: 표정, 자세
인지적 평가: 상황 해석
```

---

## 6. 정서이론 ★★★

### 제임스-랑게 이론
```
자극 → 신체반응 → 정서 경험

"우리는 울기 때문에 슬프다"

신체반응이 먼저!
```

### 캐논-바드 이론
```
자극 → 신체반응 + 정서 경험 (동시)

시상에서 동시 처리
```

### 샤흐터-싱어 이론 (2요인 이론) ★
```
자극 → 생리적 각성 + 인지적 해석 → 정서

같은 각성도 해석에 따라 다른 정서

예: 심장 뛰는 것
├── 좋아하는 사람 옆 → 설렘
└── 무서운 상황 → 두려움
```

### 비교표
```
┌─────────────────────────────────────────────────┐
│ 이론          │ 순서                          │
├─────────────────────────────────────────────────┤
│ 제임스-랑게   │ 신체반응 → 정서               │
│ 캐논-바드     │ 신체반응 = 정서 (동시)        │
│ 샤흐터-싱어   │ 각성 + 인지해석 → 정서        │
└─────────────────────────────────────────────────┘
```

---

## 7. 정서의 기능

### 적응적 기능
```
공포: 위험 회피
분노: 장애물 제거
슬픔: 사회적 지지 유발
기쁨: 행동 강화
```

### 사회적 기능
```
의사소통: 표정, 몸짓
관계 형성: 공감, 유대
```

## 핵심 암기 포인트

### 1. 매슬로 5단계
```
생리 → 안전 → 소속 → 존경 → 자아실현
(아래서 위로)
```

### 2. 정서이론 3개
```
제임스-랑게: 신체 → 정서
캐논-바드: 동시
샤흐터-싱어: 각성 + 인지
```

### 3. 내재적 vs 외재적
```
내재적: 활동 자체가 보상
외재적: 외부 보상
과잉정당화: 외적 보상이 내적 동기 ↓
```
