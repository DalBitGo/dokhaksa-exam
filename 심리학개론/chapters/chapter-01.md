# Chapter 01: 심리학의 본질

---

## 핵심 한 줄

> "심리학 = 마음과 행동을 과학적으로 연구. 관찰하고 측정할 수 있는 것만 다룸"

---

## 1. 심리학의 정의

### 심리학이란?
```
심리학 (Psychology)
= Psyche (영혼, 마음) + Logos (학문)
= 마음과 행동의 과학적 연구

과학적 = 관찰, 측정, 실험 가능
```

### 심리학의 4대 목표
```
1. 기술 (Description)
   └── "무슨 일이 일어났나?" 현상 묘사

2. 설명 (Explanation)
   └── "왜 그런가?" 원인 규명

3. 예측 (Prediction)
   └── "앞으로 어떻게 될까?" 미래 예측

4. 통제 (Control)
   └── "어떻게 바꿀 수 있나?" 변화 유도
```

---

## 2. 심리학의 역사

### 초기 심리학
```
구조주의 (분트, 1879)
├── 최초의 심리학 실험실 (독일 라이프치히)
├── 의식을 요소로 분해
├── 내성법 (자기 내면 보고)
└── "의식의 구조는 무엇인가?"

기능주의 (제임스)
├── 의식의 기능과 목적 연구
├── 다윈 진화론 영향
└── "의식은 왜 존재하는가?"
```

### 주요 학파 비교
```
┌─────────────────────────────────────────────────┐
│ 학파         │ 핵심 주장                         │
├─────────────────────────────────────────────────┤
│ 구조주의     │ 의식을 요소로 분해               │
│ 기능주의     │ 의식의 기능, 적응                │
│ 정신분석     │ 무의식이 행동 결정               │
│ 행동주의     │ 관찰 가능한 행동만               │
│ 인본주의     │ 자아실현, 긍정적 인간관          │
│ 인지심리학   │ 정보처리 과정                    │
│ 생물심리학   │ 뇌와 신경계                      │
└─────────────────────────────────────────────────┘
```

## 3. 심리학의 관점 (5대 관점)

### 1. 생물학적 관점
```
핵심: 행동 = 뇌, 신경, 유전의 결과

예: 우울증
→ 세로토닌 부족 → 약물 치료
```

### 2. 정신분석적 관점 (프로이트)
```
핵심: 행동 = 무의식적 욕구와 갈등

예: 실언 (말실수)
→ 무의식의 표출
```

### 3. 행동주의 관점 (왓슨, 스키너)
```
핵심: 행동 = 자극-반응 학습의 결과

예: 공포증
→ 과거 학습의 결과 → 조건형성으로 치료
```

### 4. 인본주의 관점 (매슬로우, 로저스)
```
핵심: 인간은 자아실현을 향해 성장

예: 낮은 자존감
→ 잠재력 실현 방해 → 자기 수용 필요
```

### 5. 인지적 관점
```
핵심: 행동 = 정보처리 방식의 결과

예: 우울한 생각
→ 부정적 인지 → 인지치료
```

## 4. 연구방법

### 연구방법 비교
```
┌───────────────────────────────────────────────┐
│ 방법      │ 특징           │ 장단점          │
├───────────────────────────────────────────────┤
│ 실험법    │ 변인 통제      │ 인과관계 O      │
│           │ 무선할당       │ 인위적 상황     │
├───────────────────────────────────────────────┤
│ 관찰법    │ 자연 상황      │ 생태적 타당도↑  │
│           │ 통제 없음      │ 인과관계 X      │
├───────────────────────────────────────────────┤
│ 조사법    │ 질문지, 면접   │ 대규모 가능     │
│           │               │ 솔직답변?       │
├───────────────────────────────────────────────┤
│ 사례연구  │ 개인 심층      │ 풍부한 정보     │
│           │               │ 일반화 어려움   │
├───────────────────────────────────────────────┤
│ 상관연구  │ 변인 간 관계   │ 관계 파악       │
│           │               │ 인과 X (상관≠인과)│
└───────────────────────────────────────────────┘
```

### 실험법 핵심 개념
```
독립변인 (IV): 연구자가 조작하는 것
종속변인 (DV): 측정되는 결과

예: "음악이 집중력에 미치는 영향"
├── IV: 음악 유무
└── DV: 집중력 점수

통제집단: 처치 안 받음
실험집단: 처치 받음
→ 비교해서 효과 확인
```

### 연구윤리
```
1. 사전 동의
2. 익명성/비밀보장
3. 해로움 최소화
4. 기만 최소화 + 사후 설명
5. 철회 권리
```

## 5. 심리학의 분야

### 기초 분야
```
├── 생리심리학: 뇌, 신경
├── 발달심리학: 생애 발달
├── 인지심리학: 사고, 기억
├── 사회심리학: 대인관계
└── 성격심리학: 개인차
```

### 응용 분야
```
├── 임상심리학: 심리치료
├── 상담심리학: 일상 적응
├── 산업/조직심리학: 직장
├── 교육심리학: 학습
└── 법정심리학: 범죄, 재판
```

## 핵심 암기 포인트

### 1. 최초의 심리학
```
분트 (1879) - 독일 라이프치히 - 구조주의 - 내성법
```

### 2. 심리학의 4대 목표
```
기술 → 설명 → 예측 → 통제
```

### 3. 5대 관점
```
생물학적 / 정신분석 / 행동주의 / 인본주의 / 인지적
```

### 4. 실험법 핵심
```
독립변인(조작) → 종속변인(측정)
실험집단 vs 통제집단
```

### 5. 상관 ≠ 인과
```
A와 B가 관련 있다 ≠ A가 B를 일으킨다
```

## 예상 문제

### Q1. 최초로 심리학 실험실을 설립한 사람은?
→ **분트** (1879, 독일 라이프치히)

### Q2. "관찰 가능한 행동만 연구 대상"이라는 관점은?
→ **행동주의**

### Q3. 독립변인과 종속변인의 차이는?
→ 독립변인은 **조작**하는 것, 종속변인은 **측정**되는 결과

### Q4. 상관연구의 한계는?
→ **인과관계를 밝힐 수 없음** (상관 ≠ 인과)
