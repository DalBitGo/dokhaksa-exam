## 핵심 암기 (자주 출제)

### 음운론 핵심
```
자음 체계 (조음 위치/방법):
├── 입술소리: ㅂㅃㅍㅁ
├── 잇몸소리: ㄷㄸㅌㄴㄹㅅㅆ
├── 센입천장: ㅈㅉㅊ
├── 여린입천장: ㄱㄲㅋㅇ
└── 목청소리: ㅎ

음운 변동:
├── 교체: ㄱ→ㅇ (국어→궁어)
├── 탈락: ㄹ탈락 (놀+는→노는)
├── 첨가: ㄴ첨가 (색연필→색년필)
└── 축약: ㅎ축약 (좋+고→조코)
```

### 품사 9품사
```
체언: 명사, 대명사, 수사
용언: 동사, 형용사
수식언: 관형사, 부사
관계언: 조사
독립언: 감탄사
```

### 문학 최초 작품들
```
최초의 신소설: 이인직 《혈의 누》
최초의 신체시: 최남선 《해에게서 소년에게》
최초의 현대시: 주요한 《불놀이》
최초의 현대소설: 이광수 《무정》
```
