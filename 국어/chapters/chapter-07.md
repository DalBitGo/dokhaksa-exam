# Chapter 07: 문학 이론

---

## 핵심 한 줄

> "문학 = 운문(시) + 산문(소설/수필). 시의 3요소, 소설의 3요소 암기 필수!"

---

## 1. 문학의 본질

### 문학의 정의
```
언어를 매체로 한 예술
사상과 감정의 표현
허구적 상상력의 산물
```

### 문학의 기능
```
쾌락적 기능: 즐거움
교훈적 기능: 가르침
종합적 기능: 쾌락 + 교훈
```

---

## 2. 문학의 갈래 ★★

### 4분법 (기본)
```
┌─────────────────────────────────────────────────┐
│ 갈래      │ 특징              │ 예시            │
├─────────────────────────────────────────────────┤
│ 서정      │ 주관적 감정       │ 시, 시조        │
│ 서사      │ 사건 전개         │ 소설, 신화      │
│ 극        │ 행동과 대화       │ 희곡, 시나리오  │
│ 교술      │ 직접 전달         │ 수필, 비평      │
└─────────────────────────────────────────────────┘
```

### 운문 vs 산문
```
운문: 운율 있음 (시, 시조, 가사)
산문: 운율 없음 (소설, 수필, 희곡)
```

---

## 3. 시의 이해 ★★★

### 시의 3요소
```
┌─────────────────────────────────────────────────┐
│ 요소      │ 내용              │ 핵심            │
├─────────────────────────────────────────────────┤
│ 운율      │ 리듬, 음악성      │ 외형률/내재율   │
│ 심상      │ 이미지, 감각      │ 시각/청각/촉각  │
│ 주제      │ 시인의 사상       │ 정서, 의미      │
└─────────────────────────────────────────────────┘
```

### 운율의 종류
```
외형률 (정형시):
├── 3·4조, 4·4조 (시조)
├── 7·5조 (개화기 시)
└── 음수율, 음보율

내재율 (자유시):
└── 반복, 대구, 호흡
```

### 심상 (이미지) ★
```
시각적 심상: 색채, 모양
청각적 심상: 소리
촉각적 심상: 감촉
후각적 심상: 냄새
미각적 심상: 맛
공감각적 심상: 감각 전이 ★
  → "분수처럼 흩어지는 푸른 종소리"
    (청각→시각)
```

### 표현 기법 ★★
```
비유법:
├── 직유: ~처럼, ~같이
├── 은유: A는 B다
├── 의인법: 사물을 사람처럼
├── 활유법: 무생물을 생물처럼
└── 대유법: 부분으로 전체 (빵=음식)

강조법:
├── 과장: 크게 부풀림
├── 반복: 같은 말 되풀이
├── 점층: 점점 강하게
└── 대조: 반대되는 것 나란히

변화법:
├── 도치: 어순 바꿈
├── 반어: 반대로 말함
├── 역설: 모순 속 진실
└── 생략: 일부 생략
```

## 4. 소설의 이해 ★★★

### 소설의 3요소
```
┌─────────────────────────────────────────────────┐
│ 요소      │ 내용              │ 핵심            │
├─────────────────────────────────────────────────┤
│ 주제      │ 작가의 의도       │ 중심 사상       │
│ 구성      │ 사건 배열         │ 플롯            │
│ 문체      │ 표현 방식         │ 문장 스타일     │
└─────────────────────────────────────────────────┘
```

### 소설의 구성 요소
```
인물 (character): 등장인물
사건 (plot): 일어나는 일
배경 (setting): 시간, 공간
```

### 구성의 5단계 ★
```
발단 → 전개 → 위기 → 절정 → 결말

발단: 인물, 배경 소개
전개: 사건 진행
위기: 갈등 심화
절정: 갈등 최고조
결말: 갈등 해소
```

### 시점 (point of view) ★★
```
1인칭 주인공 시점:
├── 나 = 주인공
└── 주관적, 독백 가능

1인칭 관찰자 시점:
├── 나 = 관찰자
└── 주인공 관찰

3인칭 관찰자 시점:
├── 그/그녀 = 주인공
└── 객관적 묘사

전지적 작가 시점:
├── 작가가 모든 것 앎
└── 내면까지 서술
```

### 갈등의 유형
```
내적 갈등: 인물 내면
외적 갈등:
├── 인물 vs 인물
├── 인물 vs 사회
└── 인물 vs 운명/자연
```

## 5. 시조와 가사

### 시조 ★
```
형식:
├── 초장 / 중장 / 종장
├── 3장 6구 45자 내외
└── 종장 첫 음절 3자 고정

종류:
├── 평시조: 기본형
├── 엇시조: 1-2구 길어짐
└── 사설시조: 많이 길어짐
```

### 가사
```
4음보 연속체
운문과 산문의 중간
교훈적, 서정적 내용
```

---

## 6. 고전 문학 핵심

### 향가
```
신라 시대 노래
향찰로 표기
4구체, 8구체, 10구체
대표작: 서동요, 처용가
```

### 고려가요
```
고려 시대 민요
3음보, 후렴구
대표작: 청산별곡, 가시리
```

### 판소리
```
1인 창자 + 고수
창 + 아니리 + 발림
판소리 다섯마당:
춘향가, 심청가, 흥부가, 수궁가, 적벽가
```

## 핵심 암기 포인트

### 1. 시의 3요소
```
운율 + 심상 + 주제
```

### 2. 소설의 3요소
```
주제 + 구성 + 문체
```

### 3. 구성 5단계
```
발단 → 전개 → 위기 → 절정 → 결말
```

### 4. 시점 4가지
```
1인칭 주인공 / 1인칭 관찰자
3인칭 관찰자 / 전지적 작가
```

## 예상 문제

### Q1. "내 마음은 호수요"에서 쓰인 표현법은?
→ **은유법** (A는 B다)

### Q2. 소설 구성의 5단계를 순서대로 쓰시오.
→ **발단 → 전개 → 위기 → 절정 → 결말**

### Q3. "분수처럼 흩어지는 푸른 종소리"에 쓰인 심상은?
→ **공감각적 심상** (청각→시각 전이)

### Q4. 작가가 모든 인물의 내면까지 아는 시점은?
→ **전지적 작가 시점**
