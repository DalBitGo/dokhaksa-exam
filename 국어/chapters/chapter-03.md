## 1. 품사 분류 기준

### 3가지 기준
```
┌─────────────────────────────────────────────────┐
│ 기준       │ 내용              │ 예시            │
├─────────────────────────────────────────────────┤
│ 형태      │ 변하냐 안 변하냐   │ 가변어/불변어    │
│ 기능      │ 문장에서 역할     │ 주어/서술어 등   │
│ 의미      │ 뜻의 성격         │ 사물/동작/상태   │
└─────────────────────────────────────────────────┘
```

### 가변어 vs 불변어
```
가변어 (활용O): 동사, 형용사
불변어 (활용X): 나머지 7품사
```

## 2. 9품사 체계 ★★★

### 전체 구조
```
┌─────────────────────────────────────────────────┐
│ 품사      │ 종류          │ 예시               │
├─────────────────────────────────────────────────┤
│ 체언      │ 명사         │ 하늘, 사람, 것      │
│ (3개)    │ 대명사       │ 나, 너, 이것, 저기   │
│          │ 수사         │ 하나, 둘, 첫째      │
├─────────────────────────────────────────────────┤
│ 관계언    │ 조사         │ 이/가, 을/를, 의    │
│ (1개)    │              │                    │
├─────────────────────────────────────────────────┤
│ 용언      │ 동사         │ 먹다, 가다, 자다    │
│ (2개)    │ 형용사       │ 예쁘다, 크다       │
├─────────────────────────────────────────────────┤
│ 수식언    │ 관형사       │ 새, 헌, 온갖       │
│ (2개)    │ 부사         │ 매우, 빨리, 과연    │
├─────────────────────────────────────────────────┤
│ 독립언    │ 감탄사       │ 아, 어머나, 네     │
│ (1개)    │              │                    │
└─────────────────────────────────────────────────┘
```

## 3. 체언 (명사, 대명사, 수사)

### 명사의 종류
```
보통명사: 사람, 책, 나무
고유명사: 한국, 서울, 김철수
자립명사: 혼자 쓰임 (하늘, 바다)
의존명사: 앞말 필요 (것, 데, 바, 줄) ★
```

### 의존명사 vs 조사 구별 ★
```
"아는 것이 힘이다"  → 것 = 의존명사 (띄어쓰기)
"먹을 수 있다"     → 수 = 의존명사
"갈 데가 없다"     → 데 = 의존명사

의존명사: 띄어 씀
조사: 붙여 씀
```

### 대명사
```
인칭대명사: 나, 너, 그, 저희
지시대명사: 이것, 그것, 저것
재귀대명사: 자기, 당신
```

---

## 4. 조사 ★★

### 조사의 종류
```
격조사: 문장성분 결정
├── 주격: 이/가, 께서, 에서
├── 목적격: 을/를
├── 관형격: 의
├── 부사격: 에, 에서, 로/으로, 와/과
└── 호격: 아/야

보조사: 특별한 의미 첨가
├── 은/는 (주제, 대조)
├── 도 (첨가)
├── 만 (한정)
├── 까지, 조차, 마저 (극단)
└── 부터 (시작점)

접속조사: 연결
└── 와/과, (이)랑, 하고
```

### 주격조사 '이/가' vs 보조사 '은/는' ★
```
이/가: 새 정보, 초점
은/는: 이미 알려진 정보, 주제

예:
"철수가 왔다" (누가 왔는지 모를 때)
"철수는 왔다" (철수를 주제로 말할 때)
```

---

## 5. 용언 (동사, 형용사)

### 동사 vs 형용사 구별 ★★
```
┌─────────────────────────────────────────────────┐
│ 구분      │ 동사              │ 형용사          │
├─────────────────────────────────────────────────┤
│ 의미      │ 동작, 작용        │ 상태, 성질      │
│ 현재형    │ -ㄴ다/는다        │ -다            │
│ 관형사형  │ -는 (현재)        │ -ㄴ/은 (현재)   │
│ 명령/청유 │ 가능              │ 불가능         │
└─────────────────────────────────────────────────┘

예:
동사: "먹는다", "먹는 밥", "먹어라"
형용사: "예쁘다", "예쁜 꽃", "예뻐라" (X)
```

### 본용언 vs 보조용언
```
본용언: 실질적 의미
보조용언: 뒤에서 의미 보조

"먹어 버리다" → 먹다(본) + 버리다(보조)
"가고 싶다"  → 가다(본) + 싶다(보조)
"읽어 보다"  → 읽다(본) + 보다(보조)
```

---

## 6. 수식언 (관형사, 부사)

### 관형사
```
성상관형사: 새, 헌, 온갖
지시관형사: 이, 그, 저
수관형사: 한, 두, 첫째

특징: 체언만 수식, 조사 안 붙음
```

### 관형사 vs 관형어 ★
```
관형사: 품사 (새 책)
관형어: 문장성분 (아름다운 꽃)

"새 책" → 새 = 관형사
"아름다운 꽃" → 아름다운 = 형용사의 관형형 (관형어)
```

### 부사
```
성분부사: 문장 일부 수식
문장부사: 문장 전체 수식

"매우 빠르다" → 매우 = 성분부사
"아마 비가 올 것이다" → 아마 = 문장부사
```

## 7. 단어의 형성

### 형태소
```
자립형태소: 홀로 가능 (산, 물)
의존형태소: 홀로 불가 (접사, 어미)

실질형태소: 실질적 의미 (어근)
형식형태소: 문법적 의미 (조사, 어미)
```

### 단어 구성
```
단일어: 형태소 1개 (하늘, 땅)
복합어: 형태소 2개 이상
├── 합성어: 어근+어근 (논밭, 오가다)
└── 파생어: 어근+접사 (풋사과, 먹이)
```

### 주요 접사 ★
```
접두사:
├── 풋- (풋사과, 풋내기): 덜 익은
├── 시- (시아버지, 시어머니): 배우자쪽
├── 헛- (헛수고, 헛소리): 보람없이
└── 맨- (맨손, 맨발): 아무것도 없이

접미사:
├── -이 (높이, 길이): 명사화
├── -음/-ㅁ (묶음, 앎): 명사화
├── -기 (먹기, 달리기): 명사화
└── -스럽다 (자랑스럽다): 형용사화
```

## 핵심 암기 포인트

### 1. 9품사 외우기
```
체언 3개: 명사, 대명사, 수사
관계언 1개: 조사
용언 2개: 동사, 형용사
수식언 2개: 관형사, 부사
독립언 1개: 감탄사
```

### 2. 동사 vs 형용사
```
동사: -ㄴ다/는다 / -는 (현재관형사형)
형용사: -다 / -ㄴ/은 (현재관형사형)
```

### 3. 조사 분류
```
격조사: 문장성분 결정 (이/가, 을/를)
보조사: 의미 첨가 (은/는, 도, 만)
```

## 예상 문제

### Q1. 다음 중 의존명사인 것은?
```
① 하늘  ② 사람  ③ 것  ④ 책
→ 정답: ③ 것 (앞말이 있어야 쓰임)
```

### Q2. "철수가 밥을 먹는다"에서 '먹는다'가 동사인 이유는?
→ **'-ㄴ다/는다' 형태로 활용** (형용사는 '-다')

### Q3. '새 책'의 '새'와 '새로운 책'의 '새로운'의 차이는?
→ **새 = 관형사(품사), 새로운 = 형용사의 관형형(활용형)**

### Q4. 보조사 '은/는'의 기능은?
→ **주제 표시, 대조**
