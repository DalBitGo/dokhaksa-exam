# Chapter 02: 음운론

---

## 핵심 한 줄

> "음운 = 말의 뜻을 구별해주는 소리. 자음/모음 체계와 음운 변동이 핵심!"

---

## 1. 음성과 음운

### 음성 vs 음운
```
음성 (Phone): 실제 발음되는 소리
음운 (Phoneme): 뜻을 구별하는 최소 단위

예:
'물'과 '불' → ㅁ과 ㅂ이 뜻 구별 → 다른 음운
'달'의 'ㄹ'과 '랄'의 'ㄹ' → 소리 달라도 같은 음운
```

### 음운의 종류
```
분절음운: 자음(19), 모음(21)
비분절음운: 장단, 억양
```

### 자음 분류 기준
```
조음 위치 (어디서):
├── 입술소리: ㅂㅃㅍㅁ
├── 잇몸소리: ㄷㄸㅌㄴㄹㅅㅆ
├── 센입천장소리: ㅈㅉㅊ
├── 여린입천장소리: ㄱㄲㅋㅇ
└── 목청소리: ㅎ

조음 방법 (어떻게):
├── 파열음: 막았다 터뜨림 (ㅂㄷㄱ)
├── 파찰음: 마찰+파열 (ㅈㅊ)
├── 마찰음: 스치듯 (ㅅㅎ)
├── 비음: 코로 나감 (ㄴㅁㅇ)
└── 유음: 혀끝 떨거나 옆 (ㄹ)
```

## 4. 음운 변동 ★★★

### 음운 변동의 유형
```
┌────────────────────────────────────────┐
│ 유형    │ 음운 수 변화  │ 예시       │
├────────────────────────────────────────┤
│ 교체    │ 유지 (1→1)   │ 국민→궁민 │
│ 탈락    │ 감소 (1→0)   │ 살+는→사는│
│ 첨가    │ 증가 (0→1)   │ 솜+이불→솜니불│
│ 축약    │ 감소 (2→1)   │ 좋+고→조코│
└────────────────────────────────────────┘
```

### 주요 음운 변동 규칙

#### 교체
```
1. 음절의 끝소리 규칙 (7종성)
   끝소리로 ㄱㄴㄷㄹㅁㅂㅇ만 발음
   예: 낮[낟], 부엌[부억], 밭[받]

2. 비음화
   ㄱㄷㅂ → ㅇㄴㅁ (비음 앞에서)
   예: 국물[궁물], 밥물[밤물]

3. 유음화
   ㄴ → ㄹ (ㄹ 앞뒤에서)
   예: 신라[실라], 난로[날로]

4. 구개음화
   ㄷㅌ + 이 → ㅈㅊ
   예: 굳이[구지], 같이[가치]
```

#### 탈락
```
1. 자음군 단순화
   겹받침 → 하나 탈락
   예: 삶[삼], 닭[닥]

2. ㅎ 탈락
   ㅎ + 모음 → ㅎ 탈락
   예: 좋아[조아]

3. ㄹ 탈락
   ㄹ + ㄴㅅㅂ → ㄹ 탈락
   예: 놀+는→노는
```

#### 첨가
```
1. ㄴ 첨가 (사이시옷 관련)
   합성어 사이에 ㄴ 첨가
   예: 색+연필→색년필, 솜+이불→솜니불

2. 반모음 첨가
   피+어→피여
```

#### 축약
```
1. 거센소리되기 (ㅎ 축약)
   ㅎ + ㄱㄷㅂㅈ → ㅋㅌㅍㅊ
   예: 좋+고→조코[조코], 넣+다→넣다[너타]

2. 모음 축약
   ㅗ+ㅏ→ㅘ, ㅜ+ㅓ→ㅝ
   예: 오+아서→와서
```

## 5. 음절 구조

### 음절의 구성
```
초성 + 중성 + 종성

초성: 자음 (0~1개)
중성: 모음 (필수)
종성: 자음 (0~1개, 발음은 7개만)
```

### 음절의 끝소리 규칙 ★
```
종성에서 발음되는 자음: 7개만
ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅇ

대표음:
ㄱ ← ㄱㅋㄲㄳㄺ
ㄴ ← ㄴㄵㄶ
ㄷ ← ㄷㅌㅅㅆㅈㅊㅎ
ㄹ ← ㄹㄼㄽㄾㅀ
ㅁ ← ㅁㄻ
ㅂ ← ㅂㅍㄿㅄ
ㅇ ← ㅇ
```

## 핵심 암기 포인트

### 1. 자음 분류
```
조음위치: 입술/잇몸/센입천장/여린입천장/목청
조음방법: 파열/파찰/마찰/비음/유음
```

### 2. 음운 변동 4유형
```
교체(1→1), 탈락(1→0), 첨가(0→1), 축약(2→1)
```

### 3. 7종성
```
ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅇ
```

### 4. 주요 규칙
```
비음화: ㄱㄷㅂ → ㅇㄴㅁ
구개음화: ㄷㅌ+이 → ㅈㅊ
유음화: ㄴ → ㄹ
```

## 예상 문제

→ [궁물] - **비음화**

### Q3. '같이'의 발음은?
→ [가치] - **구개음화**

→ **7개** (ㄱㄴㄷㄹㅁㅂㅇ)
