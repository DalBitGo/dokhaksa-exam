## 핵심 암기

### 법의 체계
```
        ┌─────────────┐
        │    헌법     │ ← 최상위
        └──────┬──────┘
               │
    ┌──────────┼──────────┐
    │          │          │
┌───┴───┐  ┌───┴───┐  ┌───┴───┐
│ 민법  │  │ 형법  │  │행정법 │
└───────┘  └───────┘  └───────┘
   │          │          │
   ↓          ↓          ↓
 개인간     국가vs개인   국가운영
 관계       (처벌)      (규제)
```

### 법의 분류
| 분류 | 구분 | 예시 |
|------|------|------|
| 공법/사법 | 공법 | 헌법, 형법, 행정법 |
| | 사법 | 민법, 상법 |
| 실체법/절차법 | 실체법 | 민법, 형법 |
| | 절차법 | 민사소송법, 형사소송법 |
| 일반법/특별법 | 일반법 | 민법 |
| | 특별법 | 상법, 노동법 |
