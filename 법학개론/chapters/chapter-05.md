# Chapter 05: 행정법

---

## 핵심 한 줄

> "행정법 = 국가(행정)와 국민의 관계. 행정이 잘못하면 행정심판/행정소송으로 구제"

---

## 1. 행정법 일반원칙

### 행정법의 특징
```
행정법 = 행정에 관한 법

민법과 다른 점:
├── 공익 우선 (사익보다)
├── 일방적 명령 가능 (평등X)
└── 행정의 공정력 (일단 유효)
```

### 행정법의 기본원리
```
1. 법치행정 원리
   ├── 법률우위: 법률 위반 X
   └── 법률유보: 법률 근거 있어야

2. 신뢰보호 원칙
   └── 행정을 믿고 한 행동 보호

3. 비례원칙 (과잉금지)
   └── 목적에 비해 수단이 과하면 X

4. 평등원칙
   └── 같은 것은 같게 취급

5. 부당결부금지 원칙
   └── 관련 없는 것 끼워넣기 X
```

## 2. 행정행위

### 행정행위란?
```
행정행위 = 행정청이 공권력으로 국민에게 하는 법적 행위

특징:
├── 일방적 (동의 없이 가능)
├── 구체적 (특정인에게)
└── 법적 효과 발생
```

### 행정행위의 종류
```
허가: 금지 해제 (영업허가)
인가: 보충적 효력 부여 (설립인가)
특허: 새 권리 설정 (광업권, 특허권 아님!)
면제: 의무 해제

명령적 행위:
├── 하명: ~하라
├── 금지: ~하지 마라
└── 허가: ~해도 된다

형성적 행위:
├── 특허: 권리 설정
├── 인가: 효력 부여
└── 대리: 대신 행위
```

### 행정행위의 효력
```
1. 공정력
   └── 권한 있는 기관이 취소할 때까지 유효

2. 불가쟁력
   └── 일정 기간 지나면 다툴 수 없음

3. 불가변력
   └── 행정청도 함부로 변경 못 함

4. 구속력
   └── 상대방, 관계기관 구속

5. 자력집행력
   └── 법원 도움 없이 직접 강제 가능
```

### 하자 있는 행정행위
```
무효 vs 취소

무효:
├── 중대하고 명백한 하자
├── 처음부터 효력 없음
└── 누구나, 언제나 주장 가능

취소:
├── 하자 있지만 중대명백 아님
├── 취소 전까지 유효
└── 취소권자만, 기간 내 주장
```

## 3. 행정강제

### 행정상 강제집행
```
대집행: 대신 하고 비용 청구
이행강제금: 돈으로 압박
직접강제: 직접 실력 행사
강제징수: 세금 등 강제 징수
```

### 행정상 즉시강제
```
급박한 경우 미리 명령 없이 강제

예: 감염병 환자 강제격리
```

## 4. 행정구제

### 행정구제 체계
```
행정작용
    │
    ├── 사전구제
    │   ├── 행정절차 (청문, 이유제시)
    │   └── 행정예고
    │
    └── 사후구제
        ├── 행정심판 (행정기관에서)
        ├── 행정소송 (법원에서)
        └── 손해배상/손실보상
```

### 행정심판
```
행정심판 = 행정기관이 심판

특징:
├── 법원 가기 전 단계 (선택적)
├── 빠르고 저렴
└── 위법 + 부당 모두 다툼 가능

종류:
├── 취소심판: 행정행위 취소 요청
├── 무효확인심판: 무효 확인 요청
└── 의무이행심판: 처분 해달라고 요청
```

### 행정소송
```
행정소송 = 법원이 재판

종류:
├── 항고소송
│   ├── 취소소송: 가장 흔함
│   ├── 무효확인소송
│   └── 부작위위법확인소송
├── 당사자소송
├── 민중소송
└── 기관소송

특징:
├── 위법성만 심사 (부당은 X)
├── 처분 등이 있음을 안 날부터 90일 내
└── 처분이 있은 날부터 1년 내
```

### 국가배상 vs 손실보상
```
국가배상 = 위법행위로 인한 손해배상
├── 공무원 직무상 불법행위
├── 영조물 설치·관리 하자
└── 국가가 배상

손실보상 = 적법행위로 인한 손실보상
├── 공익사업으로 재산권 침해
├── 정당한 보상 (헌법상 보장)
└── 예: 토지 수용
```

## 핵심 암기 포인트

### 1. 행정법 기본원리
```
법치행정 + 신뢰보호 + 비례원칙 + 평등 + 부당결부금지
```

### 2. 행정행위 효력
```
공정력: 취소 전까지 유효
불가쟁력: 기간 지나면 못 다툼
자력집행력: 직접 강제 가능
```

### 3. 무효 vs 취소
```
무효 = 중대명백 하자, 처음부터 무효
취소 = 일반 하자, 취소해야 무효
```

### 4. 행정심판 vs 행정소송
```
행정심판: 행정기관, 위법+부당
행정소송: 법원, 위법만
```

### 5. 국가배상 vs 손실보상
```
국가배상 = 위법행위 → 손해배상
손실보상 = 적법행위 → 재산권 보상
```

## 예상 문제

### Q1. 공정력이란?
→ 권한 있는 기관이 **취소하기 전까지 유효**한 것으로 취급

### Q2. 행정심판과 행정소송의 차이는?
→ 심판은 **행정기관**에서, 소송은 **법원**에서

### Q3. 국가배상의 요건은?
→ 공무원의 **위법한** 직무행위로 손해 발생

### Q4. 취소소송 제기 기간은?
→ 처분을 안 날부터 **90일**, 처분일부터 **1년**
