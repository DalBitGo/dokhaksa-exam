> "사회법 = 약자 보호 (노동법, 사회보장법). 국제법 = 나라들 간의 규칙"

---

## 1. 노동법

### 노동법의 의의
```
노동법 = 근로자 보호를 위한 법

왜 필요?
├── 고용주 vs 근로자 = 불평등
└── 국가가 개입해서 균형 맞춤
```

### 근로기준법 핵심
```
근로계약:
├── 서면 명시 (임금, 근로시간 등)
├── 위약금 예정 금지
└── 강제저금 금지

근로시간:
├── 1일 8시간, 주 40시간 원칙
├── 연장근로 한도: 주 12시간
└── 연장·야간·휴일: 50% 가산

휴가:
├── 주휴일: 1주 개근 시 1일
├── 연차: 1년 80% 이상 → 15일
└── 월차: 폐지됨 (연차로 통합)

해고:
├── 정당한 이유 필요
├── 30일 전 예고 또는 30일분 통상임금
└── 부당해고 → 노동위원회 구제신청
```

### 노동3권
```
헌법 제33조:

1. 단결권
   └── 노동조합 결성 권리

2. 단체교섭권
   └── 사용자와 교섭 권리

3. 단체행동권
   └── 파업 등 쟁의행위 권리

제한:
├── 공무원: 일부만 인정
└── 방위산업: 쟁의행위 금지
```

## 2. 사회보장법

### 사회보장 체계
```
사회보장
├── 사회보험 (보험료 내고 혜택)
│   ├── 국민연금
│   ├── 건강보험
│   ├── 고용보험
│   └── 산재보험
│
├── 공공부조 (무료 지원)
│   ├── 국민기초생활보장
│   └── 의료급여
│
└── 사회서비스
    ├── 노인복지
    ├── 장애인복지
    └── 아동복지
```

### 4대 사회보험
```
┌─────────────────────────────────────┐
│ 보험        │ 목적                  │
├─────────────────────────────────────┤
│ 국민연금    │ 노후 소득 보장        │
│ 건강보험    │ 질병, 부상 치료       │
│ 고용보험    │ 실업급여, 취업지원    │
│ 산재보험    │ 업무상 재해 보상      │
└─────────────────────────────────────┘
```

## 3. 경제법

### 경제법의 의의
```
경제법 = 경제 질서 규율

주요 법률:
├── 독점규제법 (공정거래법)
├── 소비자보호법
└── 하도급법
```

### 공정거래법 핵심
```
금지행위:
├── 시장지배적 지위 남용
├── 기업결합 제한
├── 부당 공동행위 (담합)
├── 불공정거래행위
└── 재벌 규제

담합 예:
├── 가격담합 (가격 같이 올리자)
└── 시장분할 (A지역은 내가)

제재:
├── 시정명령
├── 과징금
└── 형사처벌
```

## 4. 국제법

### 국제법의 의의
```
국제법 = 국가 간 관계 규율

국내법과 다른 점:
├── 강제 집행 어려움
├── 주권 존중 원칙
└── 국가 간 합의가 기본
```

### 국제법의 법원(法源)
```
1. 조약
   └── 국가 간 문서로 된 합의

2. 국제관습법
   └── 오랜 관행이 법이 됨

3. 법의 일반원칙
   └── 문명국가 공통 원칙

4. 판례, 학설 (보조적)
```

### 조약
```
조약 = 국가 간 합의 문서

체결 절차 (한국):
1. 협상 → 서명 (대통령)
2. 비준 → 국회 동의 필요한 것:
   ├── 입법사항
   ├── 재정부담
   ├── 국가나 국민에 중대한 영향
   └── 주권 제약
3. 공포 → 국내법과 같은 효력

헌법 제6조:
"헌법에 의하여 체결·공포된 조약과
일반적으로 승인된 국제법규는
국내법과 같은 효력"
```

### 국제기구
```
UN (국제연합):
├── 총회, 안전보장이사회
├── 상임이사국 5개국 (미중러영프)
└── 거부권

기타:
├── WTO (세계무역기구)
├── IMF (국제통화기금)
├── ICJ (국제사법재판소)
└── ICC (국제형사재판소)
```

## 5. 국제사법

### 국제사법의 의의
```
국제사법 = 섭외적 법률관계에 적용할 법 결정

예: 한국인이 미국에서 프랑스인과 계약
→ 어느 나라 법을 적용할까?

핵심 개념:
├── 준거법: 적용될 법
├── 연결점: 국적, 거소, 소재지 등
└── 국제재판관할: 어느 나라 법원?
```

## 예상 문제

### Q1. 노동3권이란?
→ **단결권, 단체교섭권, 단체행동권**

### Q2. 연장근로 시 임금 가산은?
→ 통상임금의 **50%** 이상 가산

### Q3. 조약이 국내법과 같은 효력을 가지려면?
→ **헌법에 의해 체결·공포**되어야

### Q4. 공정거래법이 금지하는 담합이란?
→ 경쟁사들끼리 **가격, 시장 분할 등을 합의**하는 행위

## 법학개론 총정리

```
법의 체계:
헌법 > 법률 > 명령 > 조례

주요 법 영역:
├── 헌법: 국가 기본, 기본권
├── 민법: 개인 간 관계
├── 형법: 범죄와 처벌
├── 행정법: 행정과 국민
├── 사회법: 약자 보호
└── 국제법: 국가 간 관계

공통 원칙:
├── 법치주의
├── 비례원칙 (과잉금지)
├── 신뢰보호
└── 적법절차
```
