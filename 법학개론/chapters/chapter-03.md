# Chapter 03: 민법/상법

---

## 핵심 한 줄

> "민법 = 개인 간 관계 규율. 권리능력(누가) + 법률행위(어떻게) + 물권/채권(무엇)"

---

## 1. 민법총칙

### 권리의 주체
```
권리능력 = 권리의무의 주체가 될 수 있는 능력
├── 자연인: 출생~사망
└── 법인: 설립등기~해산

행위능력 = 단독으로 유효한 법률행위를 할 수 있는 능력
├── 성년자 (만 19세 이상)
├── 미성년자 (만 19세 미만) → 법정대리인 동의 필요
├── 피성년후견인 → 후견인 동의 필요
└── 피한정후견인 → 중요한 행위는 동의 필요
```

### 능력 비교
```
권리능력 = "자격" (있다/없다)
의사능력 = "판단력" (이해할 수 있나)
행위능력 = "자격 + 나이" (단독으로 할 수 있나)

┌───────────────────────────────────────┐
│ 태아 → 출생 → 미성년 → 성년 → 사망  │
│       │       │        │            │
│       └─권리능력 있음──┘            │
│               │        │            │
│               └행위능력─┘            │
└───────────────────────────────────────┘
```

### 법률행위
```
법률행위 = 의사표시로 법률효과 발생

종류:
├── 단독행위: 혼자 (취소, 해제, 유언)
├── 계약: 둘이 (매매, 임대차, 고용)
└── 합동행위: 여럿이 (사단법인 설립)

성립요건:
1. 당사자
2. 목적 (적법, 가능, 확정)
3. 의사표시

효력요건:
1. 행위능력
2. 의사와 표시 일치
3. 적법, 사회적 타당성
```

### 무효와 취소
```
무효 = 처음부터 효력 없음
├── 불법, 반사회적 행위
├── 의사무능력자의 행위
└── 누구나 주장 가능

취소 = 취소하면 처음부터 무효
├── 미성년자 단독행위
├── 착오, 사기, 강박
└── 취소권자만 주장 가능
```

## 2. 물권법

### 물권이란?
```
물권 = 물건에 대한 직접적 지배권

물권 vs 채권:
┌─────────────┬─────────────┐
│    물권     │    채권     │
├─────────────┼─────────────┤
│ 물건 지배   │ 사람에게    │
│             │ 청구        │
├─────────────┼─────────────┤
│ 모든 사람에 │ 특정인에게  │
│ 대항 가능   │ 만 청구     │
├─────────────┼─────────────┤
│ 소유권, 저당│ 매매대금    │
│             │ 청구권      │
└─────────────┴─────────────┘
```

### 물권의 종류
```
물권
├── 본권 (본래의 권리)
│   ├── 소유권: 전면적 지배
│   ├── 지상권: 땅 위에 건물
│   ├── 지역권: 통행 등
│   └── 전세권: 전세
│
└── 담보물권 (빚 담보)
    ├── 유치권: 안 줄 때까지 점유
    ├── 질권: 동산 담보
    ├── 저당권: 부동산 담보
    └── 양도담보: 소유권 이전형
```

### 물권변동
```
부동산: 등기해야 효력
동산: 인도(넘겨줌)해야 효력

예: 집 매매
1. 계약 체결 → 채권 발생
2. 등기 이전 → 물권 변동 (소유권 이전)
```

## 3. 채권법

### 채권이란?
```
채권 = 특정인(채무자)에게 일정 행위를 청구할 권리

채권자 ──청구──▶ 채무자
        ◀──이행──

예: 돈 빌려줌
대여자(채권자) ──"돈 갚아"──▶ 차용자(채무자)
```

### 채권 발생 원인
```
1. 계약 (가장 많음)
   └── 매매, 임대차, 고용 등

2. 사무관리
   └── 부탁 없이 남의 일 대신함

3. 부당이득
   └── 법률상 원인 없이 이득

4. 불법행위
   └── 고의·과실로 손해 발생
       → 손해배상청구권
```

### 계약의 종류
```
쌍무계약: 양쪽 다 의무 (매매, 임대차)
편무계약: 한쪽만 의무 (증여)

유상계약: 대가 있음 (매매)
무상계약: 대가 없음 (증여)
```

### 채권의 소멸
```
1. 변제 (빚 갚음)
2. 상계 (서로 빚 있으면 상쇄)
3. 면제 (채권자가 포기)
4. 혼동 (채권자=채무자)
5. 소멸시효 (기간 지남)
```

## 4. 가족법 기초

### 혼인
```
성립요건:
├── 혼인의사 합치
├── 혼인신고 (형식)
└── 혼인 장애 없음

효과:
├── 부부공동생활
├── 부양의무
└── 재산 문제 (별산제 원칙)
```

### 상속
```
상속순위:
1순위: 직계비속 (자녀)
2순위: 직계존속 (부모)
3순위: 형제자매
4순위: 4촌 이내 방계혈족

배우자: 1,2순위와 공동상속 (1.5배)
```

---

## 5. 상법 기초

### 상법이란?
```
상법 = 상인의 영업활동 규율

민법의 특별법
→ 상인 간에는 상법 먼저
→ 없으면 민법 적용
```

### 상인의 종류
```
당연상인: 상행위를 업으로 하는 자
의제상인: 상인적 방법으로 영업
```

### 회사의 종류
```
합명회사: 무한책임사원만
합자회사: 무한 + 유한책임사원
유한회사: 유한책임사원만, 소규모
주식회사: 유한책임, 주식 발행
```

## 핵심 암기 포인트

### 1. 3가지 능력
```
권리능력: 자격 (출생~사망)
의사능력: 판단력
행위능력: 단독 가능 (성년)
```

### 2. 물권 vs 채권
```
물권 = 물건 지배, 모든 사람에게 주장
채권 = 사람에게 청구, 특정인에게만
```

### 3. 무효 vs 취소
```
무효 = 처음부터 X, 누구나 주장
취소 = 취소해야 X, 취소권자만
```

### 4. 상속순위
```
1순위 자녀 → 2순위 부모 → 3순위 형제 → 4순위 4촌
배우자는 1,2순위와 공동 (1.5배)
```

## 예상 문제

### Q1. 미성년자의 법률행위가 유효하려면?
→ **법정대리인의 동의** 필요

### Q2. 부동산 소유권 이전에 필요한 것은?
→ **등기** (등기해야 물권변동 효력)

### Q3. 불법행위가 성립하려면?
→ **고의 또는 과실** + 손해 발생 + 인과관계

### Q4. 상속에서 배우자의 상속분은?
→ 1,2순위와 공동상속, **1.5배** 가산
