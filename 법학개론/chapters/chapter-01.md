# Chapter 01: 총론 - 법의 이해

---

## 핵심 한 줄

> "법 = 사회 질서 유지하는 강제력 있는 규칙. 도덕은 양심에 맡기지만, 법은 국가가 강제함"

---

## 1. 법의 개념

### 법이란?
```
사회규범 종류:
┌──────────────────────────────────────────┐
│  도덕  │  종교  │  관습  │  ★법★        │
│        │        │        │              │
│ 양심에 │ 신앙에 │ 전통에 │ 국가가       │
│ 맡김   │ 맡김   │맡김    │ 강제함       │
└──────────────────────────────────────────┘
                              ↑
                      이게 법의 핵심!
```

### 법과 도덕의 차이
| 구분 | 법 | 도덕 |
|------|-----|------|
| 강제력 | 국가가 강제 | 양심에 맡김 |
| 외면성 | 외부 행위 규율 | 내면도 규율 |
| 명확성 | 명확한 조문 | 불명확 |
| 예시 | 살인죄 | 효도 |

### 법의 목적 (3가지)
1. **정의** - 공정한 분배, 평등
2. **법적 안정성** - 예측 가능, 일관성
3. **합목적성** - 사회 목적에 맞게

> 이 세 가지가 충돌할 때? → 대체로 정의 > 법적 안정성

---

## 2. 법의 분류

### 공법 vs 사법
```
공법 = 국가가 관여
├── 헌법 (국가 기본)
├── 형법 (처벌)
├── 행정법 (규제)
└── 소송법 (재판)

사법 = 개인 간
├── 민법 (일반적 사인 관계)
└── 상법 (상인 간 관계)
```

### 실체법 vs 절차법
```
실체법 = "무엇이" 권리/의무인지
├── 민법 (누가 돈 받을 권리?)
└── 형법 (어떤 행위가 범죄?)

절차법 = "어떻게" 실현하는지
├── 민사소송법 (민사재판 방법)
└── 형사소송법 (형사재판 방법)
```

### 일반법 vs 특별법
```
일반법 = 넓게 적용 (민법)
특별법 = 좁게 적용 (상법, 노동법)

원칙: 특별법 > 일반법
→ 상인 간 거래는 상법 먼저 적용
→ 상법에 없으면 민법 적용
```

## 3. 법원(法源) - 법의 존재 형식

### 성문법 (글로 써있는 법)
```
1. 헌법 ← 최상위
2. 법률 ← 국회가 만듦
3. 명령 ← 행정부가 만듦
   ├── 대통령령 (시행령)
   └── 부령 (시행규칙)
4. 조례 ← 지방자치단체
5. 규칙 ← 지방자치단체장
```

### 불문법 (글로 안 써있는 법)
- **관습법**: 오래된 관행이 법이 됨
- **판례법**: 대법원 판결이 기준이 됨
- **조리**: 사물의 본성, 상식

### 법 효력 순서
```
헌법 > 법률 > 명령 > 조례/규칙 > 관습법
```

---

## 4. 법의 효력

### 시간적 효력
```
원칙: 소급효 금지 (과거에 적용 X)

예외: 유리한 방향은 소급 가능
- 형벌 폐지되면 → 이전 범죄자도 석방
```

### 장소적 효력
```
원칙: 속지주의 (한국 땅에서 → 한국법)
예외: 속인주의 (한국인은 해외서도 → 한국법 일부 적용)
```

### 인적 효력
```
원칙: 모든 사람에게 평등 적용
예외: 외교관 면책특권 등
```

---

## 5. 법의 해석

### 해석 방법
| 방법 | 설명 | 예시 |
|------|------|------|
| 문리해석 | 글자 그대로 | "자동차" = 자동차만 |
| 논리해석 | 논리적으로 | 확장/축소/반대 |
| 역사적 해석 | 입법 취지 | 왜 이 법을 만들었나? |
| 체계적 해석 | 전체 맥락 | 다른 조문과 조화 |

### 유추해석 vs 확대해석
```
형법: 유추해석 금지! (죄형법정주의)
민법: 유추해석 허용
```

---

## 6. 법률관계

### 권리와 의무
```
권리 = 할 수 있음
의무 = 해야 함

예: 계약
├── 매도인: 물건 줄 의무 / 돈 받을 권리
└── 매수인: 돈 줄 의무 / 물건 받을 권리
```

### 법률행위
```
법률행위 = 의사표시로 법률효과 발생

종류:
├── 단독행위: 혼자 (유언, 취소)
├── 계약: 둘이 (매매, 임대차)
└── 합동행위: 여럿이 (사단법인 설립)
```

## 핵심 암기 포인트

### 1. 법의 3대 목적
```
정의 + 법적 안정성 + 합목적성
```

### 2. 법 분류 키워드
```
공법/사법 → 국가 개입 여부
실체법/절차법 → 무엇 vs 어떻게
일반법/특별법 → 넓게 vs 좁게
```

### 3. 법원 서열
```
헌법 > 법률 > 명령 > 조례 > 관습법
```

### 4. 형법 특수 원칙
```
- 소급효 금지
- 유추해석 금지
- 죄형법정주의
```

## 예상 문제

### Q1. 법과 도덕의 가장 큰 차이는?
→ **강제력** (국가가 강제로 실현 vs 양심에 맡김)

### Q2. "특별법 우선 원칙"이란?
→ 같은 사항에 대해 일반법과 특별법이 있으면 **특별법을 먼저 적용**

### Q3. 형법에서 유추해석이 금지되는 이유는?
→ **죄형법정주의** (법에 없는 행위를 처벌하면 안 됨)

### Q4. 법률이 헌법에 위반되면?
→ **무효** (헌법이 최상위법)
