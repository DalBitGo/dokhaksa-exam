## 1. 형법총론

### 죄형법정주의
```
"법률 없으면 범죄 없고, 법률 없으면 형벌 없다"

핵심 내용:
├── 관습형법 금지 (법률로 명확히)
├── 소급효 금지 (과거 행위 처벌 X)
├── 유추해석 금지 (확대 적용 X)
└── 명확성 원칙 (애매하면 안 됨)
```

### 범죄 성립 요건 (3요소)
```
범죄 = 구성요건해당성 + 위법성 + 책임

1. 구성요건해당성
   └── 법에서 정한 행위 유형에 해당?
   └── 예: "사람을 죽인 자" → 살인죄

2. 위법성
   └── 정당한 이유 없이 법질서 위반?
   └── 정당방위면 위법성 없음

3. 책임
   └── 비난 가능한가?
   └── 심신장애면 책임 감경/면제
```

### 위법성 조각사유
```
위법성 조각 = 위법한 것 같지만 합법

1. 정당방위
   └── 현재의 부당한 침해 방어

2. 긴급피난
   └── 위난 피하려고 타인에게 손해

3. 자구행위
   └── 권리 보전 위해 즉시 행동

4. 피해자 승낙
   └── 피해자가 동의함

5. 정당행위
   └── 법령, 업무, 사회상규
```

### 책임조각사유
```
책임 조각 = 비난할 수 없음

1. 형사미성년자 (만 14세 미만)
2. 심신장애자
3. 강요된 행위
```

### 미수범
```
예비 → 착수 → 실행 → 기수
       │              │
       └── 미수 ──────┘

미수 = 시작했지만 완성 못 함
├── 장애미수: 외부 장애로 못 함
├── 중지미수: 자발적으로 그만둠 (감경)
└── 불능미수: 처음부터 불가능
```

### 공범
```
공범 = 여러 명이 범죄

1. 공동정범: 함께 실행 (모두 정범)
2. 교사범: 시킴
3. 방조범: 도움
```

### 형벌의 종류
```
주형:
├── 사형
├── 징역 (교도소 노역)
├── 금고 (교도소 구금만)
├── 자격상실
├── 자격정지
├── 벌금
├── 구류 (30일 미만)
└── 과료 (5만원 미만)

부가형:
├── 몰수
└── 추징
```

## 2. 형법각론 (주요 범죄)

### 개인적 법익
```
생명:
├── 살인죄
└── 자살관여죄

신체:
├── 상해죄
├── 폭행죄
└── 과실치사상죄

자유:
├── 체포·감금죄
├── 협박죄
└── 강요죄

명예:
├── 명예훼손죄
└── 모욕죄
```

### 재산범죄
```
┌─────────────────────────────────────┐
│ 재산범죄 분류                        │
├─────────────────────────────────────┤
│ 탈취 (빼앗음)                        │
│ ├── 절도: 몰래 가져감               │
│ ├── 강도: 폭행·협박으로 빼앗음       │
│ └── 횡령: 맡은 물건 착복            │
├─────────────────────────────────────┤
│ 편취 (속임)                          │
│ └── 사기: 속여서 재물 취득          │
├─────────────────────────────────────┤
│ 손괴 (망가뜨림)                      │
│ └── 손괴죄                          │
└─────────────────────────────────────┘
```

### 절도 vs 강도 vs 횡령 vs 사기
```
절도: 남의 물건을 몰래 가져감
강도: 폭행/협박 + 절도
횡령: 맡아둔 물건을 내 것처럼
사기: 속여서 스스로 주게 함
배임: 업무상 신임을 배반하여 손해
```

## 3. 형사소송법

### 형사절차 흐름
```
범죄발생 → 수사 → 기소 → 공판 → 판결 → 집행

수사:
├── 수사기관: 검찰, 경찰
├── 강제수사: 영장 필요
└── 임의수사: 동의로 가능

기소:
├── 공소제기: 검사만 가능 (기소독점)
├── 기소편의주의: 기소 여부 재량
└── 불기소처분: 무혐의, 기소유예 등
```

### 수사의 원칙
```
1. 영장주의
   └── 강제수사는 법관 영장 필요

2. 적법절차
   └── 위법수집증거 배제

3. 무죄추정
   └── 유죄 확정 전까지 무죄로 추정
```

### 영장의 종류
```
체포영장: 사람 체포
구속영장: 사람 구금
압수·수색영장: 물건 압수, 장소 수색

예외 (영장 없이):
├── 현행범 체포
├── 긴급체포 (중죄)
└── 긴급압수·수색
```

### 피의자의 권리
```
1. 진술거부권 (묵비권)
2. 변호인 조력권
3. 접견교통권
4. 영장제시 요구권
```

```
구성요건해당성 + 위법성 + 책임
```
