## 1. 헌법의 기본원리

### 대한민국 헌법의 5대 원리
```
┌────────────────────────────────────────┐
│         대한민국 헌법 기본원리          │
├────────────────────────────────────────┤
│ 1. 국민주권주의 - 권력은 국민에게      │
│ 2. 자유민주주의 - 자유 + 민주          │
│ 3. 복지국가원리 - 국가가 복지 책임     │
│ 4. 문화국가원리 - 문화 존중            │
│ 5. 국제평화주의 - 침략전쟁 부인        │
└────────────────────────────────────────┘
```

### 헌법 전문(前文) 키워드
- 3·1운동, 대한민국임시정부
- 민주개혁

- 자율과 조화

## 2. 기본권

### 기본권 체계
```
기본권
├── 포괄적 기본권
│   └── 인간의 존엄과 가치, 행복추구권
│
├── 자유권 (국가로부터 자유)
│   ├── 신체의 자유
│   ├── 거주·이전의 자유
│   ├── 직업선택의 자유
│   ├── 주거의 자유
│   ├── 사생활의 자유
│   ├── 통신의 자유
│   ├── 양심의 자유
│   ├── 종교의 자유
│   ├── 언론·출판의 자유
│   └── 집회·결사의 자유
│
├── 평등권 (차별 금지)
│   └── 법 앞의 평등
│
├── 참정권 (정치 참여)
│   ├── 선거권
│   ├── 피선거권
│   └── 국민투표권
│
├── 청구권 (국가에 요구)
│   ├── 청원권
│   ├── 재판청구권
│   ├── 국가배상청구권
│   ├── 형사보상청구권
│   └── 범죄피해자구조청구권
│
└── 사회권 (국가에 적극 요청)
    ├── 교육을 받을 권리
    ├── 근로의 권리
    ├── 환경권
    └── 인간다운 생활을 할 권리
```

### 기본권 제한 (과잉금지원칙)
```
기본권 제한의 4가지 요건:

1. 목적의 정당성
   └── 왜 제한해야 하나?

2. 방법의 적정성
   └── 그 방법이 적절한가?

3. 피해의 최소성
   └── 덜 제한하는 방법은 없나?

4. 법익의 균형성
   └── 얻는 것 > 잃는 것?

모두 충족해야 합헌!
```

### 기본권 제한의 한계
```
제37조 2항:
"국가안전보장, 질서유지, 공공복리를 위해
필요한 경우에 한하여
법률로써 제한 가능.
본질적 내용은 침해 불가"
```

## 3. 통치구조 (3권 분립)

### 전체 구조
```
┌──────────────────────────────────────────────┐
│              국  민 (주권자)                  │
└──────────────────┬───────────────────────────┘
                   │ 선거
    ┌──────────────┼──────────────┐
    │              │              │
    ▼              ▼              ▼
┌───────┐    ┌──────────┐    ┌───────┐
│ 국회  │    │ 대통령   │    │ 법원  │
│(입법) │◀──▶│ (행정)   │◀──▶│(사법) │
└───────┘    └──────────┘    └───────┘
법 만듦      법 집행         법 해석

    ↑                           ↑
    └───────────────────────────┘
           서로 견제 (Check)
```

### 국회 (입법부)
```
구성: 국회의원 300명, 임기 4년

주요 권한:
├── 입법권 (법률 제정)
├── 재정권 (예산 심의)
├── 국정조사권
├── 탄핵소추권
├── 국무총리 임명동의권
└── 조약 비준 동의권

의결 정족수:
├── 일반: 재적 과반 출석, 출석 과반 찬성
└── 특별: 헌법개정, 탄핵 등은 가중
```

### 대통령 (행정부)
```
임기: 5년 단임 (연임 불가)

주요 권한:
├── 행정권 수반
├── 국군 통수권
├── 공무원 임명권
├── 사면권
├── 긴급명령권
├── 법률안 거부권
└── 국민투표 부의권

탄핵:
├── 국회 재적 2/3 찬성으로 소추
└── 헌법재판소 재판관 6인 찬성으로 결정
```

### 법원 (사법부)
```
구조:
대법원 ← 최고법원 (대법관 14명)
   ↑
고등법원
   ↑
지방법원

대법원장:
├── 임기 6년
├── 국회 동의 얻어 대통령이 임명
└── 연임 불가
```

## 4. 헌법재판소

### 헌법재판소 vs 대법원
```
헌법재판소         vs         대법원
─────────────────────────────────────
헌법 관련 심판              일반 재판 최종
9명 (대통령 3,               14명
국회 3, 대법원장 3)
```

### 헌법재판소 권한
```
1. 위헌법률심판
   └── "이 법률 헌법 위반?" 판단

2. 탄핵심판
   └── 대통령, 법관 등 탄핵

3. 정당해산심판
   └── 위헌 정당 해산

4. 권한쟁의심판
   └── 국가기관 간 분쟁

5. 헌법소원심판
   └── 기본권 침해 구제
```

### 결정 정족수
```
위헌결정, 탄핵결정, 정당해산:
→ 재판관 6인 이상 찬성 필요
```

## 5. 헌법 개정

### 개정 절차
```
1. 제안
   ├── 국회 재적 과반수
   └── 또는 대통령

2. 공고
   └── 20일 이상 공고

3. 국회 의결
   └── 재적 2/3 이상 찬성

4. 국민투표
   └── 투표자 과반 찬성

5. 공포
   └── 대통령이 공포
```

## 핵심 암기 포인트

### 1. 기본권 분류
```
자유권 = 국가가 간섭하지 마
참정권 = 정치에 참여할래
청구권 = 뭔가 해달라고 요구
사회권 = 복지 보장해줘
```

### 2. 과잉금지원칙 (4가지)
```
목적 정당 + 방법 적정 + 피해 최소 + 법익 균형
```

### 3. 3권분립
```
입법(국회) - 행정(대통령) - 사법(법원)
→ 서로 견제하여 독재 방지
```

### 4. 헌법재판소 핵심 숫자
```
9명, 6년 임기, 6인 이상 찬성
```

## 예상 문제

### Q1. 기본권 제한의 한계는?
→ **본질적 내용 침해 금지** + 과잉금지원칙 준수

### Q2. 헌법재판소와 대법원의 차이는?
→ 헌재는 **헌법 관련 심판**, 대법원은 **일반 재판 최종심**

### Q3. 대통령 탄핵에 필요한 국회 의결 정족수는?
→ 재적의원 **2/3 이상** 찬성

### Q4. 자유권과 사회권의 차이는?
→ 자유권은 **국가 간섭 배제**, 사회권은 **국가 적극 개입 요청**
