# 독학사 시험 준비 프로젝트

## 현재 상황

- **시험**: 독학사 1단계 (2026년 3월 8일)
- **목표**: 60점 이상 합격 (과락 없이 전 과목)
- **과목**: 법학개론, 심리학개론, 국어, 영어, 한국사

---

## 공부 방법: Active Learning

### 핵심 원칙

```
1. 소크라테스식 질문으로 진짜 이해 확인
2. 단순 암기 X → "왜?"를 아는 학습
3. Q&A 세션으로 능동적 recall
```

### 질문 유형

사용자가 챕터를 공부할 때 다음 질문들로 이해도 확인:

```
- "이게 뭐야?" → 핵심 개념
- "왜 필요해?" → 존재 이유
- "없으면 어떻게 돼?" → 중요성
- "언제/어디서 쓰여?" → 적용
- "비슷한 거랑 뭐가 달라?" → 구분
- "결국 뭐냐면?" → 한 문장 정리
```

### 세션 진행 방식

1. **챕터 선택**: 사용자가 공부할 챕터 선택
2. **내용 확인**: 해당 챕터 읽고 핵심 파악
3. **Q&A 진행**: 소크라테스식 질문으로 테스트
4. **약점 파악**: 틀린 부분 재확인
5. **예상문제 풀이**: 챕터 하단 문제 활용

## 챕터 파일 구조

각 챕터는 다음 형식으로 구성됨:

```markdown
# Chapter XX: 제목

## 핵심 한 줄
> "한 문장 요약"

## 1~N. 내용 섹션
(ASCII 표, 구조화된 설명)

## 핵심 암기 포인트
(시험에 나오는 것만)

## 예상 문제
Q1. 질문
→ 정답
```

## 사용자 요청 시 행동

### "공부하자" / "복습하자"

1. 어떤 과목/챕터 할지 물어보기
2. 해당 챕터 읽기
3. Q&A 세션 시작 (소크라테스식)

### "테스트해줘" / "문제 내줘"

1. 챕터의 예상문제 활용
2. 추가로 변형 문제 출제
3. 틀린 부분 설명

### "이거 설명해줘"

1. "결국 뭐냐면" 스타일로 핵심만
2. 왜 중요한지 설명
3. 관련 예시 제공
