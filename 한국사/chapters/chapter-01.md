# Chapter 01: 선사시대와 고조선

---

## 핵심 한 줄

> "구석기→신석기→청동기→철기. 고조선은 청동기 문화 기반, 8조법으로 사회상 파악!"

---

## 1. 구석기 시대

### 특징
```
시기: 약 70만 년 전 시작
생활: 이동생활 (동굴, 막집)
경제: 수렵, 채집, 어로
도구: 뗀석기 (주먹도끼, 찍개, 슴베찌르개)
사회: 무리생활, 평등

시기별 도구:
├── 전기: 주먹도끼, 찍개
└── 후기: 슴베찌르개
```

### 주요 유적지
```
단양 금굴 (최초 발견)
공주 석장리
연천 전곡리 (아슐리안 주먹도끼)
```

---

## 2. 신석기 시대

### 특징
```
시기: 약 1만 년 전 ~ BC 2000년
생활: 정착생활 시작 (움집)
경제: 농경 시작, 목축
도구: 간석기, 빗살무늬토기
사회: 씨족사회, 평등

신석기 혁명:
└── 농경 시작 → 정착 → 사회 변화
```

### 빗살무늬토기
```
밑이 뾰족함 (땅에 꽂아 세움)
서울 암사동 대표 유적
곡물 저장에 사용
```

### 신앙
```
애니미즘: 정령 숭배
토테미즘: 동식물 숭배
샤머니즘: 무당을 통해 소통
조상숭배
```

---

## 3. 청동기 시대

### 특징
```
시기: BC 2000년 ~ BC 400년
생활: 농경 본격화 (벼농사)
도구: 청동기 (의기, 무기), 민무늬토기
사회: 계급 발생 (군장)

청동기 = 지배층 전용 (농기구 X)
실제 농사에는 간석기 사용
```

### 대표 유물
```
비파형동검: 요령~한반도
세형동검: 한반도 특화형
거친무늬거울 (다뉴조문경)
미송리식토기, 민무늬토기
```

### 고인돌
```
지배층의 무덤
분포: 전 세계 절반이 한반도
├── 탁자식 (북방식): 북부
└── 바둑판식 (남방식): 남부
```

### 사회 변화
```
잉여 생산물 → 사유재산 → 계급 발생
군장 (족장) 등장 → 정복전쟁 → 국가 형성
```

## 4. 고조선

### 건국
```
건국자: 단군왕검
건국연대: BC 2333년 (삼국유사)
건국지역: 요령 + 한반도 북부
수도: 평양 (이동: 아사달 → 평양)
```

### 단군신화의 의미
```
환인(하늘) - 환웅 - 단군

의미:
├── 선민사상 (하늘의 자손)
├── 농경사회 (풍백, 우사, 운사 = 바람, 비, 구름)
├── 홍익인간 (널리 이롭게)
└── 제정일치
```

### 8조법 ★
```
현재 3개조만 전해짐:

1. 사람을 죽인 자는 사형
2. 상해를 입힌 자는 곡물로 배상
3. 도둑질한 자는 노비로 삼음
   (속죄하려면 50만 전)

의미:
├── 생명 존중
├── 사유재산 인정
├── 계급 (노비) 존재
└── 형벌체계 존재
```

### 위만조선 (BC 194)
```
위만: 연나라 출신, 망명
준왕을 몰아내고 왕위 찬탈
철기 보급 → 중계무역 발달
한과 대립 → BC 108년 멸망
→ 한사군 설치
```

## 5. 여러 나라의 성장

### 부여
```
위치: 만주 송화강 유역
정치: 왕 + 마가, 우가, 저가, 구가 (사출도)
풍습: 영고 (12월 제천행사)
       순장, 1책12법
형벌: 살인→사형, 도둑→12배 배상
```

### 고구려
```
위치: 압록강 유역
정치: 왕 + 5부족 연맹
풍습: 동맹 (10월 제천행사)
       서옥제 (데릴사위제)
       형사취수 (형 죽으면 형수와 결혼)
```

### 옥저/동예
```
옥저:
├── 민며느리제 (어린 신부)
└── 가족공동무덤

동예:
├── 무천 (10월 제천행사)
├── 책화 (경계 침범 배상)
└── 특산물: 단궁, 과하마, 반어피
```

### 삼한
```
위치: 한반도 남부
구성: 마한, 진한, 변한

마한 → 백제로 발전
진한 → 신라로 발전
변한 → 가야로 발전

풍습:
├── 소도 (제사장 천군, 신성지역)
├── 5월 수릿날, 10월 추수감사제
└── 벼농사, 철 생산 (변한)
```

## 핵심 암기 포인트

### 1. 시대별 도구
```
구석기: 뗀석기 (주먹도끼)
신석기: 간석기, 빗살무늬토기
청동기: 비파형동검, 민무늬토기
```

### 2. 제천행사
```
부여: 영고 (12월)
고구려: 동맹 (10월)
동예: 무천 (10월)
삼한: 5월, 10월
```

### 3. 8조법 의미
```
생명 존중, 사유재산 인정, 계급 존재
```

### 4. 삼한 → 삼국
```
마한→백제, 진한→신라, 변한→가야
```

## 예상 문제

### Q1. 신석기 시대의 대표적인 토기는?
→ **빗살무늬토기**

### Q2. 계급 사회가 시작된 시대는?
→ **청동기 시대**

### Q3. 8조법에서 알 수 있는 사회상은?
→ **사유재산 인정, 계급(노비) 존재**

### Q4. 고구려의 제천행사는?
→ **동맹** (10월)
