# Chapter 04: 조선 전기

---

## 핵심 한 줄

> "태조 건국 → 세종 문화 → 성종 경국대전. 과전법, 집현전, 훈구/사림 대립 기억!"

---

## 1. 조선 건국

### 건국 과정
```
위화도 회군 (1388): 이성계, 권력 장악
과전법 실시 (1391): 경제적 기반 마련
조선 건국 (1392): 태조 이성계
한양 천도 (1394)
```

### 태조의 정책
```
정치: 정도전의 재상 중심 설계
경제: 과전법 (관리에게 토지)
사상: 숭유억불
```

---

## 2. 왕권 강화

### 태종
```
6조 직계제: 왕에게 직접 보고
호패법: 신분 확인, 유민 방지
사병 폐지: 왕권 강화
```

### 세조
```
직전법: 현직자에게만 토지
집현전 폐지
단종 폐위 (계유정난)
```

---

## 3. 세종의 업적 ★★★

### 정치/제도
```
집현전 설치: 학술 연구
의정부 서사제: 재상 합의제
```

### 문화/과학
```
훈민정음 창제 (1443, 반포 1446)
칠정산: 역법
측우기: 세계 최초 강우량 측정
앙부일구: 해시계
자격루: 물시계
농사직설: 우리 농법
```

### 국방
```
4군 6진 개척 (압록강~두만강)
대마도 정벌
```

---

## 4. 통치체제 ★

### 중앙 정치
```
┌─────────────────────────────────────────────────┐
│ 기관       │ 역할                              │
├─────────────────────────────────────────────────┤
│ 의정부     │ 최고 회의 기관                    │
│ 6조        │ 행정 실무 (이호예병형공)          │
│ 3사        │ 언론/감찰                         │
│ ├ 사헌부   │ 관리 감찰                        │
│ ├ 사간원   │ 왕에게 간쟁                      │
│ └ 홍문관   │ 경연, 자문                       │
│ 승정원     │ 왕명 출납                        │
│ 의금부     │ 왕명 수행, 특별 사법              │
└─────────────────────────────────────────────────┘
```

### 지방 제도
```
8도 (관찰사 파견)
부-목-군-현
향리: 지방 실무
유향소: 향촌 자치
```

### 신분제도
```
양반: 문반 + 무반
중인: 기술관, 서얼
상민: 농민, 상인, 수공업자
천민: 노비
```

---

## 5. 경국대전 ★

### 완성 과정
```
세조: 경국대전 편찬 시작
성종 (1485): 경국대전 완성, 반포
```

### 특징
```
조선 최고 법전
6전 체제: 이·호·예·병·형·공전
이후 법전의 기준
```

---

## 6. 훈구와 사림

### 훈구파
```
세조 때 공신
권력 독점, 대토지 소유
향촌에서 유향소 장악
```

### 사림파
```
영남 출신 (김종직 계열)
성리학 중시
향약, 서원 통해 향촌 장악
성종 때 중앙 진출
```

### 사화 ★
```
무오사화 (1498): 김종직 제자 탄압
갑자사화 (1504): 연산군 (폐비 윤씨)
기묘사화 (1519): 중종 (조광조)
을사사화 (1545): 명종 (외척 대립)

암기: "무갑기을" 사화
```

---

## 7. 붕당정치

### 붕당의 시작
```
이조전랑 문제 (선조)
동인 (김효원) vs 서인 (심의겸)
```

### 붕당의 분화
```
동인 → 남인 / 북인
서인 → 노론 / 소론

인조반정 이후 서인 집권
```

## 핵심 암기 포인트

### 1. 조선 초기 왕 업적
```
태조: 건국, 과전법
태종: 6조 직계제, 호패법
세종: 훈민정음, 집현전, 4군 6진
세조: 직전법, 집현전 폐지
성종: 경국대전 완성
```

### 2. 4대 사화
```
무오(1498)-갑자(1504)-기묘(1519)-을사(1545)
```

### 3. 3사의 기능
```
사헌부: 감찰
사간원: 간쟁
홍문관: 자문, 경연
```

## 예상 문제

### Q1. 조선 최고 법전은?
→ **경국대전** (성종 때 완성)

### Q2. 훈민정음을 창제한 왕은?
→ **세종**

### Q3. 조광조가 죽은 사화는?
→ **기묘사화** (1519)

### Q4. 조선의 삼사(三司)가 아닌 것은?
→ 승정원 (3사 = 사헌부, 사간원, 홍문관)
