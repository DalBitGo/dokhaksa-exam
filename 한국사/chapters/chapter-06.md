## 1. 흥선대원군의 개혁 (1863-1873)

### 왕권 강화
```
┌─────────────────────────────────────────────┐
│ 정책            │ 내용                      │
├─────────────────────────────────────────────┤
│ 비변사 폐지     │ 의정부 기능 회복          │
│ 삼군부 부활     │ 군사권 장악               │
│ 대전회통 편찬   │ 법전 정비                 │
│ 서원 철폐       │ 47개만 남김 (붕당 타파)   │
│ 경복궁 중건     │ 왕권 상징 (원납전, 당백전)│
└─────────────────────────────────────────────┘
```

### 통상 수교 거부 (쇄국정책)
```
병인양요 (1866): 프랑스 침략 → 격퇴
신미양요 (1871): 미국 침략 → 격퇴
척화비: "서양 오랑캐와 싸우지 않고 화친하면 나라를 파는 것"
```

## 2. 개항과 불평등 조약

### 강화도 조약 (1876) ★★★
```
최초의 근대적 조약 + 불평등 조약

불평등 내용:
├── 치외법권 (영사재판권)
├── 해안 측량권
├── 무관세
└── 개항장 (부산, 원산, 인천)

의의:
└── 조선 문호 개방, 열강 침탈 시작
```

### 조미수호통상조약 (1882)
```
최혜국 대우 조항 포함
거중조정 조항 (분쟁 조정)
```

## 3. 개화운동과 갈등

### 개화파의 분화
```
온건 개화파 (동도서기)     급진 개화파 (전면개화)
├── 김홍집, 김윤식         ├── 김옥균, 박영효
├── 점진적 개혁            ├── 급진적 개혁
├── 청 의존                ├── 일본 모델
└── 갑오개혁 주도          └── 갑신정변 주도
```

### 임오군란 (1882)
```
원인: 신식군대(별기군) 우대, 구식군대 차별
주체: 구식 군인 + 하층민
결과:
├── 대원군 일시 재집권
├── 청의 개입 → 대원군 납치
└── 청의 내정 간섭 강화
```

### 갑신정변 (1884) ★★
```
주체: 김옥균, 박영효 (급진 개화파)
계기: 우정국 개국 축하연
3일 천하로 실패

14개조 개혁안:
├── 청과 사대관계 폐지
├── 문벌 폐지, 인민 평등
├── 지조법 개혁
├── 내시부 폐지
└── 순사 제도 도입

실패 원인:
├── 청군 개입
├── 민중 기반 부족
└── 일본의 미온적 태도
```

## 4. 동학농민운동 (1894) ★★★

### 전개 과정
```
고부 농민 봉기 (1894.1)
├── 전봉준 주도
└── 고부 군수 조병갑 횡포

1차 봉기: 반봉건
├── 황토현 전투 승리
├── 전주성 점령
└── 전주화약 (정부와 타협)

폐정개혁안 12개조:
├── 탐관오리 처벌
├── 노비문서 소각
├── 토지 균분
└── 외세 배척

집강소 설치:
└── 농민 자치기구

2차 봉기: 반외세
├── 일본군 침략에 맞서 재봉기
├── 공주 우금치 전투 패배
└── 전봉준 체포, 처형
```

### 의의
```
반봉건 + 반외세 운동
갑오개혁에 영향
이후 항일 의병운동의 기반
```

## 5. 갑오개혁과 대한제국

### 갑오개혁 (1894-1896)
```
1차 개혁 (군국기무처):
├── 개국 연호 사용
├── 과거제 폐지
├── 신분제 폐지
└── 은본위제

2차 개혁 (김홍집 내각):
├── 홍범 14조 반포
├── 교육입국조서
├── 재판소 설치
└── 태양력 사용
```

### 대한제국 (1897)
```
선포: 1897년 고종
연호: 광무
황제국 표방 (자주독립)
구본신참 (옛것 지키며 새것 참작)

광무개혁:
├── 양전사업 (토지 조사)
├── 지계 발급
├── 상공업 진흥
└── 한계: 황제권 강화 중심
```

## 6. 일제 침략

### 러일전쟁과 침략
```
1904 러일전쟁
├── 한일의정서
└── 제1차 한일협약 (외교 고문)

1905 을사늑약 ★
├── 외교권 박탈
├── 통감부 설치 (이토 히로부미)
└── 고종의 헤이그 특사 파견

1907 한일신협약
├── 차관 정치
├── 군대 해산
└── 고종 강제 퇴위

1910 경술국치
└── 한일병합조약 (나라 잃음)
```

### 국권회복운동
```
의병운동:
├── 을미의병 (1895, 단발령)

├── 정미의병 (1907, 군대해산)
└── 13도 창의군 (서울 진공 작전)

애국계몽운동:
├── 보안회, 헌정연구회
├── 대한자강회
├── 신민회 (안창호, 비밀결사)
└── 국채보상운동 (1907, 대구)
```

## 7. 일제강점기 (1910-1945)

### 통치 방식 변화
```
┌─────────────────────────────────────────────────┐
│ 시기        │ 특징           │ 계기             │
├─────────────────────────────────────────────────┤
│ 1910년대    │ 무단통치       │ 헌병경찰제       │
│             │               │ 조선태형령       │
├─────────────────────────────────────────────────┤
│ 1920년대    │ 문화통치       │ 3·1운동 이후     │
│             │ (기만적)       │ 보통경찰제       │
├─────────────────────────────────────────────────┤
│ 1930-45년   │ 민족말살통치   │ 침략전쟁 확대    │
│             │               │ 황국신민화       │
└─────────────────────────────────────────────────┘
```

### 3·1 운동 (1919) ★★
```
배경: 윌슨의 민족자결주의, 2·8 독립선언
전개:
├── 민족대표 33인 독립선언
├── 탑골공원 → 전국 확산
└── 비폭력 → 무력 저항

의의:
├── 최대 규모 항일운동
├── 대한민국 임시정부 수립
└── 일제 통치방식 변화 (무단→문화)
```

### 대한민국 임시정부 (1919)
```
수립: 상하이
제도: 민주공화제, 대통령제
초대 대통령: 이승만
활동:
├── 연통제, 교통국 (국내 연결)
├── 독립신문 발간
├── 한국광복군 창설 (1940)
└── 대일 선전포고 (1941)
```

## 핵심 암기 포인트

### 1. 근대 주요 사건 순서
```
1876 강화도조약
1882 임오군란
1884 갑신정변
1894 동학농민운동, 갑오개혁
1897 대한제국
1905 을사늑약
1910 경술국치
1919 3·1운동, 임시정부
```

### 2. 갑신정변 vs 동학
```
갑신정변: 급진개화파, 위로부터, 3일
동학: 농민, 아래로부터, 반봉건+반외세
```

### 3. 일제 통치 3단계
```
무단(1910s) → 문화(1920s) → 민족말살(1930-45)
```
