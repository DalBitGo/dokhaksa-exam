## 1. 고려 건국과 후삼국 통일

### 건국
```
918년: 왕건, 고려 건국 (송악=개경)
935년: 신라 항복 (경순왕)
936년: 후백제 멸망 → 후삼국 통일
```

### 태조 왕건의 정책 ★
```
호족 통합:
├── 사성정책: 왕씨 성 하사
├── 기인제도: 호족 자제 볼모
├── 혼인정책: 호족과 결혼
└── 역분전: 공신에게 토지

훈요 10조:
├── 불교 숭상
├── 서경(평양) 중시
├── 거란(차령 이북 사람) 경계
└── 신하의 간언 수용
```

## 2. 통치체제 정비

### 광종의 개혁 ★★
```
왕권강화:
├── 노비안검법: 불법 노비 해방
├── 과거제 실시: 유교 교양
├── 공복제정: 관료 복색 구분
└── 독자 연호 (광덕, 준풍)
```

### 성종의 정비
```
유교 정치:
├── 최승로의 시무 28조
├── 12목에 지방관 파견
├── 국자감 설치 (교육)
└── 2성 6부제
```

### 중앙 정치 조직
```
도병마사/식목도감: 고려 독자적 기구 (합좌)
중서문하성: 정책 결정
상서성: 정책 집행
중추원: 군사 기밀
어사대: 감찰
```

## 3. 과거제와 신분제도 ★

### 과거제
```
종류:
├── 제술과: 문학 (진사과)
├── 명경과: 유교 경전
└── 잡과: 기술직

특징:
├── 양인 이상 응시 가능
├── 음서와 병존
└── 무과는 없었다가 후기에 생김
```

### 음서제도
```
5품 이상 관리 자손에게 관직 부여
과거 없이 관직 진출
```

### 신분제도
```
귀족: 문벌 귀족 (과거+음서)
중류층: 남반, 잡류, 향리
양민: 농민, 상인
천민: 노비, 화척, 재인
```

## 5. 무신정권 (1170-1270)

### 원인과 전개
```
원인:
├── 문신 우대, 무신 차별
├── 의종의 사치, 무신 멸시
└── 보현원 사건 (정변 직접 계기)

전개:
├── 정중부 → 경대승 → 이의민
└── 최씨 정권 (4대 60년)
    최충헌 → 최우 → 최항 → 최의
```

### 최씨 정권
```
교정도감: 최고 권력기구
정방: 인사권 장악
삼별초: 사병 집단
```

### 민중 봉기
```
망이·망소이의 난 (공주)
만적의 난 (노비 해방 시도)
```

## 6. 원 간섭기

### 간섭 내용
```
정치:
├── 정동행성 설치
├── 관제 격하 (성→사, 부→감)
├── 왕 호칭 '충' + 종/왕
└── 다루가치 파견

수탈:
├── 공녀, 환관 징발
├── 매사냥용 응방
└── 일본 원정 부담
```

### 공민왕의 개혁 ★★
```
반원 자주:
├── 정동행성 이문소 폐지
├── 쌍성총관부 수복
├── 관제 복구
└── 몽골풍 금지

내정 개혁:
├── 전민변정도감: 권문세족 토지 환수
├── 신돈 등용
└── 성균관 정비

좌절: 권문세족 반발, 홍건적·왜구 침입
```

## 7. 고려 문화

### 불교
```
국가적 행사: 연등회, 팔관회
종파: 교종 (화엄종) + 선종
의천: 천태종 창시 (교관겸수)
지눌: 조계종 (선교일치, 정혜쌍수)
```

### 역사서
```
삼국사기: 김부식, 유교사관, 현존 최고(古)
삼국유사: 일연, 불교사관, 단군 수록
```

### 예술
```
청자: 순청자 → 상감청자 ★
금속활자: 직지심체요절 (현존 최고)
팔만대장경: 해인사
```
