# Chapter 02: 삼국시대와 남북국시대

---

## 핵심 한 줄

> "고구려(북방강국) + 백제(문화전파) + 신라(삼국통일). 남북국 = 통일신라 + 발해!"

---

## 1. 삼국의 성립

### 고구려 (BC 37)
```
건국: 주몽 (졸본)
수도: 국내성 → 평양
전성기: 광개토대왕, 장수왕
특징: 북방 강국, 중국과 대결
```

### 백제 (BC 18)
```
건국: 온조 (위례성)
수도: 한성 → 웅진 → 사비
전성기: 근초고왕
특징: 해상 교역, 일본에 문화 전파
```

### 신라 (BC 57)
```
건국: 박혁거세 (서라벌)
수도: 경주
전성기: 진흥왕
특징: 삼국 중 가장 늦게 발전, 삼국통일
```

---

## 2. 삼국의 발전 ★★

### 고구려 전성기
```
광개토대왕 (391-413):
├── 영토 확장 (만주, 요동)
├── 백제 공격, 신라 구원
└── 광개토대왕릉비 (업적 기록)

장수왕 (413-491):
├── 평양 천도
├── 남하정책 (백제 한성 점령)
└── 고구려 최대 영토
```

### 백제 전성기
```
근초고왕 (346-375):
├── 마한 통합
├── 고구려 고국원왕 전사시킴
├── 일본에 칠지도 하사
└── 가야 진출
```

### 신라 전성기
```
진흥왕 (540-576):
├── 한강 유역 차지
├── 화랑도 국가 조직화
├── 황초령, 마운령 순수비
└── 대가야 정복
```

---

## 3. 삼국의 문화 ★

### 불교
```
고구려: 소수림왕 (372) - 순도
백제: 침류왕 (384) - 마라난타
신라: 법흥왕 (527) - 이차돈 순교

신라가 가장 늦음 (귀족 반발)
```

### 율령/교육
```
고구려: 소수림왕 율령반포, 태학
백제: 오경박사
신라: 법흥왕 율령반포, 화랑도
```

### 고분
```
고구려: 돌무지무덤 → 굴식돌방무덤 (벽화)
백제: 굴식돌방무덤 (무령왕릉)
신라: 돌무지덧널무덤 (금관, 천마총)
```

---

## 4. 삼국통일 ★★

### 통일 과정
```
660년: 백제 멸망 (나당연합군)
668년: 고구려 멸망 (나당연합군)
676년: 당 축출 (매소성, 기벌포 전투)
       → 신라 삼국통일 완성
```

### 통일의 의의와 한계
```
의의:
├── 최초의 민족 통일
├── 자주적 (당 축출)
└── 문화 융합

한계:
├── 대동강~원산만 이남
└── 고구려 영토 상실
```

---

## 5. 통일신라

### 통치체제
```
왕권강화:
├── 9주 5소경 (지방제도)
├── 9서당 10정 (군사제도)
└── 집사부 (최고 관청)
```

### 골품제 ★
```
┌─────────────────────────────────────────────────┐
│ 골품      │ 관등 제한    │ 해당              │
├─────────────────────────────────────────────────┤
│ 성골      │ 왕족         │ 진덕여왕까지      │
│ 진골      │ 1~5등        │ 왕족, 최고귀족    │
│ 6두품     │ 6등 아찬까지 │ 중간 귀족         │
│ 5~1두품   │ 그 아래      │ 일반              │
└─────────────────────────────────────────────────┘

골품제의 한계 → 6두품 불만 → 신라 멸망 원인
```

### 불교 문화
```
불국사, 석굴암 (경덕왕 시기)
에밀레종 (성덕대왕신종)
```

---

## 6. 발해 ★

### 건국과 발전
```
건국: 대조영 (698, 동모산)
      → 고구려 유민 + 말갈
민족: 지배층=고구려 유민

전성기: 선왕 (9세기 초)
├── 해동성국 (당에서 부름)
├── 5경 15부 62주
└── 당, 일본과 교류
```

### 발해의 문화
```
고구려 계승:
├── 온돌
├── 이불병좌상 (불상)
└── 정혜공주묘 (굴식돌방)

이원적 통치:
├── 지배층: 고구려식
└── 피지배층: 말갈식
```

### 남북국 시대
```
신라 (남) + 발해 (북)
= 남북국 시대

발해 멸망: 926년 (거란)
```

## 핵심 암기 포인트

### 1. 삼국 건국 순서
```
신라(BC 57) → 고구려(BC 37) → 백제(BC 18)
※ 발전은 고구려 → 백제 → 신라 순
```

### 2. 불교 전래 순서
```
고구려(372) → 백제(384) → 신라(527)
```

### 3. 삼국통일 과정
```
백제(660) → 고구려(668) → 당 축출(676)
```

### 4. 발해 = 고구려 계승
```
건국: 대조영 (고구려 유민)
전성: 선왕 (해동성국)
```

## 예상 문제

### Q1. 한강 유역을 차지하고 순수비를 세운 왕은?
→ **진흥왕** (신라)

### Q2. 통일신라의 신분제도는?
→ **골품제** (성골-진골-두품)

### Q3. 발해를 '해동성국'이라 불렀을 때의 왕은?
→ **선왕** (전성기)

### Q4. 삼국 중 불교가 가장 늦게 공인된 나라는?
→ **신라** (법흥왕 527년)
