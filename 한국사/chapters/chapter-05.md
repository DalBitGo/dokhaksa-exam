# Chapter 05: 조선 후기

---

## 핵심 한 줄

> "양난 극복 → 영·정조 탕평책 → 세도정치 → 개항. 대동법, 균역법, 실학자 암기!"

---

## 1. 임진왜란 (1592-1598) ★★

### 전개
```
1592: 일본 침략 (부산 상륙)
조선군 패퇴: 선조 의주 피난

반격:
├── 이순신: 한산도대첩 (학익진)
├── 권율: 행주대첩
└── 의병: 곽재우, 고경명, 조헌

1597: 정유재란
1598: 이순신 전사 (노량해전), 왜군 철수
```

### 영향
```
국내:
├── 인구 격감, 경작지 감소
├── 국가 재정 악화
├── 신분제 동요 (공명첩)
└── 문화재 소실

일본: 도공 납치 → 도자기 발달
명: 국력 소모 → 후금 성장
```

---

## 2. 병자호란 (1636)

### 원인과 전개
```
원인:
├── 인조반정 (친명정책)
└── 후금(청) 성장

전개:
├── 청 태종 침입
├── 인조 남한산성 항전
└── 삼전도 굴욕 (항복)

결과:
├── 소현세자, 봉림대군 볼모
├── 북벌론 대두
└── 조선 = 청의 조공국
```

---

## 3. 제도 개편 ★★

### 대동법 ★
```
배경: 방납의 폐단
내용: 특산물 대신 쌀(포/전) 납부
실시: 광해군~숙종 (100년 걸림)
의의: 공인 등장, 상품화폐경제 발달
```

### 균역법
```
배경: 군포 2필 부담 과중
내용: 군포 1필로 감면
보완: 결작, 선무군관포, 어염선세
시기: 영조
```

### 영정법
```
내용: 전세 영구 고정 (1결당 4두)
시기: 인조
```

---

## 4. 탕평정치 ★

### 영조
```
탕평비 건립
이조전랑 권한 약화
균역법 실시
속대전, 동국문헌비고
서얼 허통 (청요직 진출 허용)
```

### 정조
```
규장각 설치: 왕권 강화
장용영 설치: 친위부대
수원 화성 건설
서얼 허통 확대
대전통편, 문체반정
```

---

## 5. 세도정치 (1800-1863)

### 특징
```
왕권 약화
외척 가문 권력 독점:
├── 순조: 안동 김씨
├── 헌종: 풍양 조씨
└── 철종: 안동 김씨

삼정 문란:
├── 전정: 토지세 폐단
├── 군정: 군포 폐단
├── 환곡: 환곡 폐단
```

### 농민 봉기
```
홍경래의 난 (1811): 평안도 차별
진주 농민 봉기 (1862): 삼정 문란
→ 전국으로 확산
```

## 6. 실학 ★★

### 중농학파 (경세치용)
```
유형원: 반계수록, 균전론
이익: 성호사설, 한전론
정약용: 목민심서, 여전론
        경세유표, 흠흠신서
        거중기 설계
```

### 중상학파 (이용후생)
```
유수원: 우서
홍대용: 의산문답, 지전설
박지원: 열하일기, 양반전, 허생전
박제가: 북학의, 소비 진작
```

### 국학
```
역사:
├── 이익: 성호사설
├── 안정복: 동사강목
└── 유득공: 발해고 (남북국 시대)

지리:
├── 이중환: 택리지
├── 김정호: 대동여지도 ★

한글:
├── 신경준: 훈민정음운해
```

## 7. 조선 후기 문화

### 서민 문화
```
판소리, 탈춤
한글 소설: 홍길동전, 춘향전
풍속화: 김홍도, 신윤복
진경산수화: 정선 (인왕제색도)
```

### 경제 변화
```
농업:
├── 이앙법 (모내기) 확대
├── 광작 (부농 성장)
└── 상품작물 재배

상업:
├── 사상 (선대제 상인) 성장
├── 개시, 후시 (대청 무역)
├── 금속화폐 유통 (상평통보)
└── 장시 발달
```

## 핵심 암기 포인트

### 1. 양난
```
임진왜란 (1592): 이순신, 의병
병자호란 (1636): 삼전도 굴욕
```

### 2. 제도 개편
```

균역법: 군포 2필→1필
```

```
유형원-반계수록 / 이익-성호사설

```

