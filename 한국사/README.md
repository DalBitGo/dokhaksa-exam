## 시대별 핵심 암기

### 선사시대
```
구석기: 뗀석기, 주먹도끼
신석기: 간석기, 빗살무늬토기, 농경시작
청동기: 비파형동검, 민무늬토기, 계급발생
철기: 철제농기구, 고조선 발전
```

### 고조선
```
건국: 단군왕검 (BC 2333)
범위: 요령~한반도 북부
8조법: 생명, 재산, 노비 중시
위만조선 → 한의 침략 (BC 108)
```

### 삼국 왕권 강화
```
고구려: 소수림왕 (불교, 태학, 율령)
백제: 근초고왕 (전성기)
신라: 법흥왕 (불교, 율령, 금관가야 병합)
```

### 근대 주요 사건
```
1876 강화도조약 (최초 근대 조약, 불평등)
1882 임오군란 (구식군인)
1884 갑신정변 (급진 개화파)
1894 동학농민운동, 갑오개혁
1897 대한제국 선포
1905 을사늑약
1910 경술국치
```
