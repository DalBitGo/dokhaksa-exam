## Q&A

### Q1: 헌법이 결국 뭐야? 왜 필요해?

**"국가 권력 못 믿어서 만든 족쇄"**

```
옛날: 왕이 맘대로 → 국민 고통
지금: 헌법으로 권력 제한 → 국민 보호

헌법 = 국가한테 거는 족쇄
     = 국민 기본권 보장 + 권력 분립
```

핵심 2가지:
```
1. 기본권 보장
   → 국가가 함부로 못 건드림

2. 권력 분립 (3권분립)
   → 입법/행정/사법 나눠서 견제
   → 독재 방지
```

**헌법 = 국민 보호 + 독재 방지**

---

### Q2: 기본권 종류 5가지

```
1. 자유권 = "국가야 간섭하지 마"
   → 신체, 종교, 언론, 직업선택 등
   → 국가로부터 자유

2. 평등권 = "차별하지 마"
   → 법 앞에 평등

3. 참정권 = "나도 정치 참여할래"
   → 선거권, 피선거권, 국민투표권

4. 청구권 = "국가야 해결해줘"
   → 재판청구권, 국가배상청구권

5. 사회권 = "국가야 보장해줘"
   → 교육, 근로, 환경, 인간다운 생활
```

**시험 포인트:**
```
자유권 vs 사회권

자유권: 국가가 "안" 건드리는 것 (소극적)
사회권: 국가가 "해주는" 것 (적극적)
```
