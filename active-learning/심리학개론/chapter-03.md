## Q&A

### Q1: 피아제가 말하는 "인지발달"이 뭐야?

**결국 피아제가 말하고 싶은 건:**
"애들 머리는 어른 머리 축소판이 아니다"

```
아이 → 청소년 → 어른

단순히 지식이 쌓이는 게 아니라
"생각하는 방식 자체"가 단계별로 바뀜

2살: 눈앞에 없으면 없는 거
5살: 내가 보면 다 보이는 거
8살: 모양 바뀌어도 양은 같네
15살: 가설 세우고 추론 가능
```

**인지발달 = 생각하는 방식이 질적으로 변화하는 과정**

### Q2: 피아제 4단계가 뭐야?

```
1. 감각운동기 (0~2세)
   → 입에 넣고, 만지고, 던지면서 세상 파악
   → 핵심: "대상영속성" - 눈앞에서 사라져도 있다는 걸 앎
   → 까꿍놀이 재밌는 이유

2. 전조작기 (2~7세)
   → 말은 하는데 논리는 아직
   → 핵심 3개:
      • 자기중심성: 내가 보면 다 보이는 줄 앎
      • 물활론: 인형이 아파한다고 생각
      • 보존개념 X: 물컵 모양 바뀌면 양도 바뀐 줄 앎

3. 구체적 조작기 (7~11세)
   → 논리 생김, 근데 눈앞에 있는 것만
   → 핵심: "보존개념 O" - 모양 바뀌어도 양은 같다

4. 형식적 조작기 (11세~)
   → 추상적 사고 가능
   → "만약에 ~라면?" 가설 세우고 추론
```

**시험 포인트:**
```
전조작기 = 자기중심성, 물활론, 보존개념 X
구체적 조작기 = 보존개념 O
```
