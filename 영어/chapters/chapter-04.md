# Chapter 04: 문법 심화

---

## 핵심 한 줄

> "준동사(to부정사/동명사/분사) + 관계사 + 가정법 = 고득점 열쇠!"

---

## 1. 준동사 개요

### 준동사란?
```
동사에서 파생, 동사 아닌 역할
동사의 성질 유지 (목적어, 보어 취함)

종류:
├── to부정사: to + 동사원형
├── 동명사: V-ing (명사 역할)
└── 분사: V-ing / V-ed (형용사 역할)
```

---

## 2. to부정사 ★★★

### 3가지 용법
```
명사적 용법: 주어/목적어/보어
├── To study is important. (주어)
├── I want to go. (목적어)
└── My dream is to travel. (보어)

형용사적 용법: 명사 수식
├── I need a book to read.
└── She has no time to waste.

부사적 용법: 목적/결과/원인
├── I came to help you. (목적)
├── He grew up to be a doctor. (결과)
└── I'm happy to see you. (원인)
```

### to부정사만 목적어로 취하는 동사 ★
```
want, hope, wish, expect, decide,
plan, promise, refuse, agree, learn

암기: "원하고 희망하고 기대하고 결정하면 to부정사"
```

### 가주어-진주어 구문
```
It is + 형용사 + to V

It is important to study.
= To study is important.

It takes + 시간 + to V
It takes 2 hours to get there.
```

---

## 3. 동명사 ★★★

### 동명사의 역할
```
명사처럼 사용 (주어/목적어/보어)

주어: Swimming is fun.
목적어: I enjoy reading.
보어: My hobby is cooking.
전치사 뒤: I'm good at swimming.
```

### 동명사만 목적어로 취하는 동사 ★
```
enjoy, finish, mind, avoid, give up,
keep, consider, suggest, deny, quit

암기: "megapeps" - mind, enjoy, give up, avoid,
      practice, escape, postpone, suggest
```

### to부정사 vs 동명사 의미 차이 ★★
```
remember to V: 앞으로 할 일 기억
remember V-ing: 과거에 한 일 기억

forget to V: 할 일을 잊다
forget V-ing: 했던 것을 잊다

stop to V: ~하려고 멈추다
stop V-ing: ~하는 것을 멈추다

try to V: ~하려고 노력하다
try V-ing: 시험 삼아 ~하다

예:
I remembered to lock the door. (잠글 것을 기억함)
I remember locking the door. (잠근 것을 기억함)
```

---

## 4. 분사 ★★

### 현재분사 vs 과거분사
```
현재분사 (V-ing): 능동, 진행
과거분사 (V-ed): 수동, 완료

The boring movie (지루하게 하는)
The bored audience (지루해진)

exciting news (흥분시키는)
excited people (흥분된)
```

### 감정 동사 분사 ★
```
┌─────────────────────────────────────────────────┐
│ 원인 (사물)      │ 결과 (사람)              │
├─────────────────────────────────────────────────┤
│ interesting      │ interested              │
│ boring           │ bored                   │
│ exciting         │ excited                 │
│ surprising       │ surprised               │
│ confusing        │ confused                │
│ disappointing    │ disappointed            │
└─────────────────────────────────────────────────┘
```

### 분사구문
```
시간/이유/조건 표현

Being tired, I went to bed early.
= Because I was tired, ~

Walking down the street, I met Tom.
= While I was walking, ~
```

## 5. 관계사 ★★★

### 관계대명사
```
선행사가 사람: who (주격), whom (목적격), whose (소유격)
선행사가 사물: which (주격/목적격), whose/of which (소유격)
사람/사물 둘 다: that

예:
The man who lives here is my uncle.
The book which I bought is interesting.
The girl whose father is a doctor.
```

### 관계대명사 that 선호 ★
```
선행사에 다음이 올 때 that 선호:
├── 최상급: the best that
├── 서수: the first that
├── the only, the very, the same
├── all, every, no, any
└── 사람 + 사물: The man and his dog that~
```

### 관계대명사 생략
```
목적격 관계대명사는 생략 가능

The book (which) I bought
The man (whom) I met
```

### 관계부사
```
선행사가 장소: where (= in/at which)
선행사가 시간: when (= at/on which)
선행사가 이유: why (= for which)
선행사가 방법: how (= the way) ※ 둘 중 하나만

예:
This is the place where I was born.
= This is the place in which I was born.

I know the reason why he left.
This is how I did it. (the way I did it)
```

### what vs that ★
```
what: 선행사 포함 (~하는 것)
that: 선행사 필요

What I need is money.
= The thing that I need is money.

I don't know what he said.
I know the fact that he left.
```

## 6. 가정법 ★★★

### 가정법 과거 (현재 사실 반대)
```
If + S + 과거동사, S + would/could + 동사원형

If I were rich, I would travel.
(실제: 나는 부자가 아니다)

If I had time, I could help you.
(실제: 나는 시간이 없다)

주의: be동사는 인칭 관계없이 were
If I were you, ~ (내가 너라면)
```

### 가정법 과거완료 (과거 사실 반대)
```
If + S + had p.p., S + would have p.p.

If I had studied harder, I would have passed.
(실제: 열심히 공부하지 않았다)

If it had not rained, we could have gone.
(실제: 비가 왔다)
```

### 혼합 가정법
```
과거 조건 → 현재 결과

If I had studied, I would be rich now.
(과거에 공부했다면, 지금 부자일 텐데)
```

### I wish + 가정법
```
I wish + 과거: 현재 소망
I wish I were taller. (키가 더 컸으면)

I wish + 과거완료: 과거 후회
I wish I had studied harder. (더 열심히 했더라면)
```

### as if / as though
```
마치 ~인 것처럼

He talks as if he knew everything.
(실제로 모르지만)

She looked as if she had seen a ghost.
(실제로 본 것은 아니지만)
```

## 핵심 암기 포인트

### 1. to부정사 vs 동명사 목적어
```
to부정사: want, hope, decide, plan, promise
동명사: enjoy, finish, mind, avoid, give up
```

### 2. 분사 선택
```
사물 주어 → -ing (boring movie)
사람 주어 → -ed (bored person)
```

### 3. 관계사 선택
```
사람: who/whom/whose
사물: which/whose
둘다: that
선행사 포함: what
```

### 4. 가정법 시제
```
가정법 과거: 과거형 → 현재 반대
가정법 과거완료: had p.p. → 과거 반대
```

## 예상 문제

### Q1. "I enjoy ______ books."의 빈칸에 알맞은 것은?
→ **reading** (enjoy는 동명사 목적어)

### Q2. "The ______ news surprised everyone."
→ **surprising** (news가 원인)

### Q3. "If I ______ you, I would accept the offer."
→ **were** (가정법 과거, be동사는 were)

### Q4. "This is the house ______ I was born."
→ **where** 또는 **in which**
