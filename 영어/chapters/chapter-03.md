## 1. 품사와 문장 성분

### 8품사
```
┌─────────────────────────────────────────────────┐
│ 품사      │ 역할           │ 예시              │
├─────────────────────────────────────────────────┤
│ 명사      │ 이름           │ book, Korea       │
│ 대명사    │ 명사 대신      │ he, it, this      │
│ 동사      │ 동작/상태      │ run, is           │
│ 형용사    │ 명사 수식      │ beautiful, big    │
│ 부사      │ 동/형/부 수식  │ very, quickly     │
│ 전치사    │ 명사 앞        │ in, on, at        │
│ 접속사    │ 연결           │ and, but, because │
│ 감탄사    │ 감정 표현      │ oh, wow           │
└─────────────────────────────────────────────────┘
```

### 5형식
```
1형식: S + V
   The sun rises.

2형식: S + V + C (주격보어)
   She is beautiful.

3형식: S + V + O
   I love you.

4형식: S + V + IO + DO (간접목적어 + 직접목적어)
   He gave me a book.

5형식: S + V + O + OC (목적격보어)
   I found the book interesting.
```

## 2. 시제 (Tense) ★★★

### 12시제 체계
```
┌───────────────────────────────────────────────────┐
│           │ 단순       │ 진행          │ 완료      │
├───────────────────────────────────────────────────┤
│ 현재      │ do/does    │ am/is/are     │ have      │
│           │            │ doing         │ done      │
├───────────────────────────────────────────────────┤
│ 과거      │ did        │ was/were      │ had done  │
│           │            │ doing         │           │
├───────────────────────────────────────────────────┤
│ 미래      │ will do    │ will be       │ will have │
│           │            │ doing         │ done      │
└───────────────────────────────────────────────────┘
```

### 현재시제
```
용법:
├── 현재의 상태: I am a student.
├── 반복적 습관: I exercise every day.
├── 불변의 진리: The earth is round.
├── 왕래발착 동사 (미래): The train leaves at 5.
└── 시간/조건절: If it rains, I will stay.
```

### 과거시제
```
용법:
├── 과거의 사실: I went to school.
├── 역사적 사실: Columbus discovered America.
└── 과거의 습관: I used to smoke. (used to)
```

### 현재완료 (have + p.p.) ★
```
용법:
├── 완료: I have just finished.
├── 경험: I have been to Paris.
├── 계속: I have lived here for 10 years.
└── 결과: I have lost my key. (지금 없음)

함께 쓰이는 부사:
├── 완료: just, already, yet
├── 경험: ever, never, before
└── 계속: for, since
```

### 과거완료 (had + p.p.)
```
대과거 (과거보다 더 과거):

Before I arrived, he had left.
(내가 도착했을 때, 그는 이미 떠났다)

과거───────────────────현재
      ↑         ↑
   had left   arrived
```

## 3. 수동태 (Passive Voice)

### 기본 형태
```
능동: S + V + O
수동: O + be + p.p. (+ by S)

예:
능동: Tom broke the window.
수동: The window was broken by Tom.
```

### 시제별 수동태
```
현재: is/am/are + p.p.
과거: was/were + p.p.
미래: will be + p.p.
현재완료: have been + p.p.
진행: is being + p.p.
```

### 4형식의 수동태
```
능동: He gave me a book.
수동1: I was given a book (by him).
수동2: A book was given to me (by him).
```

### 5형식의 수동태
```
능동: They elected him president.
수동: He was elected president.
```

## 4. 조동사 (Modal Verbs)

### 주요 조동사
```
┌─────────────────────────────────────────────────────┐
│ 조동사    │ 의미              │ 예문                │
├─────────────────────────────────────────────────────┤
│ can       │ 능력/가능         │ I can swim.         │
│ could     │ 과거 능력/가능성  │ It could be true.   │
├─────────────────────────────────────────────────────┤
│ may       │ 허락/추측         │ It may rain.        │
│ might     │ 약한 추측         │ It might work.      │
├─────────────────────────────────────────────────────┤
│ must      │ 의무/강한 추측    │ You must study.     │
│ have to   │ 의무 (외부)       │ I have to go.       │
├─────────────────────────────────────────────────────┤
│ should    │ 충고/당위         │ You should rest.    │
│ ought to  │ = should          │ You ought to go.    │
├─────────────────────────────────────────────────────┤
│ will      │ 의지/미래         │ I will do it.       │
│ would     │ 과거 의지/정중    │ Would you help me?  │
└─────────────────────────────────────────────────────┘
```

### 추측의 강도
```
확신 강함 ──────────────────▶ 확신 약함

must > will > would > should > can > may > might
(틀림없이)                              (혹시)
```

### 조동사 + have p.p.
```
must have p.p.: ~했음에 틀림없다
should have p.p.: ~했어야 했다 (후회)
could have p.p.: ~할 수 있었다 (안 함)
may have p.p.: ~했을지도 모른다
```

## 5. 수 일치 (Agreement)

### 기본 규칙
```
단수 주어 → 단수 동사
복수 주어 → 복수 동사

He runs. (단수)
They run. (복수)
```

### 주의할 경우
```
1. 수식어구에 속지 말 것:
   The books on the table [are] mine.
   (주어는 books)

2. Every, Each + 단수:
   Every student [has] a book.
   Each of them [is] ready.

3. 부분 표현 (some, most, all):
   앞의 명사에 일치
   Some of the water [is] cold.
   Some of the books [are] old.

4. 주어가 and로 연결:
   Tom and Jerry [are] friends.

   예외: 하나의 개념
   Bread and butter [is] my breakfast.
```

## 핵심 암기 포인트

### 1. 현재완료 키워드
```
완료(just, already) / 경험(ever, never) / 계속(for, since)
```

### 2. 수동태 공식
```
be + p.p. (by 행위자)
```

### 3. 조동사 추측 강도
```
must > may > might
```

### 4. 수 일치 핵심
```
주어 찾기! (수식어에 속지 말 것)
```

## 예상 문제

### Q1. "I have lived here for 10 years"의 시제는?
→ **현재완료** (계속 용법)

### Q2. 능동문 "They built the house"의 수동태는?
→ **The house was built (by them).**

### Q3. "must have done"의 의미는?
→ **~했음에 틀림없다** (과거 추측)

### Q4. "The news is good"에서 news가 단수인 이유는?
→ **news는 항상 단수 취급**
