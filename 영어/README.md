## 핵심 문법 정리

### 시제
```
현재: do/does (습관, 진리)
과거: did (과거 사실)
미래: will do
현재완료: have p.p. (경험, 완료, 계속)
과거완료: had p.p. (대과거)
```

### 준동사
```
to부정사: to + 동사원형
├── 명사적: 주어/목적어/보어
├── 형용사적: 명사 수식
└── 부사적: 목적/결과/원인

동명사: V-ing (명사 역할)

분사:
├── 현재분사 V-ing: 능동/진행
└── 과거분사 V-ed: 수동/완료
```

### 관계사
```
관계대명사:
├── who(m): 사람
├── which: 사물
├── that: 사람/사물
└── whose: 소유

관계부사:
├── where: 장소
├── when: 시간
├── why: 이유
└── how: 방법
```

### 가정법
```
가정법 과거: If S + 과거동사, S + would/could + 동사원형
→ 현재 사실의 반대

가정법 과거완료: If S + had p.p., S + would have p.p.
→ 과거 사실의 반대

혼합가정: 과거 조건 + 현재 결과
```

## 자주 출제되는 어휘

### 고난도 어휘
```
implement: 실행하다
affluent: 부유한
competent: 유능한
substantial: 상당한
reluctant: 꺼리는
inevitable: 불가피한
prominent: 저명한
adequate: 충분한
```

### 자주 헷갈리는 단어
```
affect (동) vs effect (명)
principal (주요한) vs principle (원칙)
complement (보완) vs compliment (칭찬)
stationary (정지한) vs stationery (문구류)
```
