## 학습 방법

### 1. kb 스타일 = 체계적 정리
- 각 과목별 챕터 구조
- 핵심 개념 위주 정리
- "결국 뭐냐면" 스타일

### 2. active-learning 스타일 = Q&A 학습
- 소크라테스식 질문
- "왜 필요해?", "없으면 어떻게 돼?"
- 틀린 것 위주로 기록
